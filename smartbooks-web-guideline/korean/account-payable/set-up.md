# 2.1.	입력

외상매입금은 두 가지 주요 기능이 있습니다.

·         입력 (Input)

·         보고서 (Reports)

<figure><img src="../.gitbook/assets/image (106).png" alt=""><figcaption></figcaption></figure>

### 2.1.       입력

#### 2.1.1.   매입영수증

재고이동과 관련된 영수증 입력의 경우, 재고관리 모듈과 연결됩니다.

원재료, 소모품 및 외부 서비스 요금(전기비, 수도비, 전화비, 사무실 임차료, 등)과 관련된 매입 영수증에 대해 입력이 가능합니다.

<figure><img src="../.gitbook/assets/image (107).png" alt=""><figcaption></figcaption></figure>

**매입영수증 입력 세부절차 안내**

**신규를** 선택하여 입력 화면을 엽니다.

<figure><img src="../.gitbook/assets/image (108).png" alt=""><figcaption></figcaption></figure>

·         일반내역 (General area)

&#x20;         o   모듈전표번호(Voucher No.): 전표번호 입력 또는 소프트웨어가 자동 기입.

&#x20;         o   전표일자(Voucher Date): 영수증 날짜는 회계기간이내여야 합니다.

&#x20;         o   기간(mm-yyyy): 소프트웨어가 전표일자를 기준으로 자동 기입합니다.

&#x20;         o   세금계산서 번호/시리얼 번호 / 세금계산서 발행일: 세금계산서에 표시된 정보입니다.

&#x20;         o   매입처(Vendor ID): F3키를 눌러 매입처 목록에서 선택.

&#x20;         o   외상매입금 계정과목: F3키를 눌러 목록에서 선택 또는 상세 계좌 번호를 입력

&#x20;         o   통화 유형: 외화를 선택하여 환율을 입력합니다.

&#x20;         o   지급 방식: : F3키를 눌러 목록에서 선택

&#x20;         o   지급일자: 지급일자 선택

&#x20;         o   적요(Description, VN-EN): 적요 입력

<figure><img src="../.gitbook/assets/image (109).png" alt=""><figcaption></figcaption></figure>

·         **상세내역(Detail area)**

영수증의 세부 내역을 나타냅니다. 아래의 데이터 시트 부분에 입력이 가능합니다.

&#x20;         o   계정과목(Account): F3키를 눌러 비용계정 선택

&#x20;         o   통화 유형: 지급에 사용될 통화 유형을 선택합니다. (소프트웨어는 일반내역에서 기본값으로 기입합니다).

&#x20;         o   환율: 선택한 통화의 VND 대비 환율을 입력합니다. (소프트웨어는 일반내역에서 기본값으로 기입합니다).

&#x20;         o   금액: 세금 제외 금액 또는 외화 금액을 입력합니다.

&#x20;         o   환산금액: 입력한 금액에 환율을 곱한 후의 금액입니다.

&#x20;         o   세율(Tax rate): F3 키를 눌러 해당 세율 유형을 선택합니다

&#x20;         o   세금 금액은 실제 발생한 세금 금액입니다.

&#x20;         o   환산 세금 금액은 (있을 경우) 환율을 곱한 후의 금액입니다.

&#x20;         o   적요(Description, VN-EN): 일반내역에 입력한 내용을 자동으로 기입합니다.

&#x20;         o   세금계산서 번호/ 시리얼 번호/ 세금계산서 발행일: 세금계산서에 표시된 정보를 입력합니다.

&#x20;

#### 2.1.2.   모듈전표 마감

본 항목은 외상매입금 모듈에 입력된 전표를 총계정원장에 전기하는 기능을 수행합니다. 전기할 항목 모두 또는 월별로 선택하여 “실행”버튼을 누릅니다.

·         모듈전표를 개별적으로 마감합니다.

&#x20;         o   전표 목록에서 마감할 모듈전표를 선택합니다.

&#x20;         o   “마감”를 선택합니다.

·         모듈전표를 일괄적으로 마감합니다.

&#x20;         o   목록에서 마감되지 않은 전표(보류 상태)를 필터링합니다.

&#x20;         o   마감할 전표를 선택합니다.

&#x20;         o   “마감 완료”를 선택합니다.
