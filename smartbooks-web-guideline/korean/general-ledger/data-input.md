# 1.2.	입력

#### 1.2.1.   전표 입력

총계정원장 모듈 사용시, 일반 거래 또는 기타 특정거래를 직접 입력이 가능하니다.

데이터 입력 안내

·         **신규를** 선택하여 입력 창을 엽니다.

<figure><img src="../.gitbook/assets/image (87).png" alt=""><figcaption></figcaption></figure>

·         분개 내역:

&#x20;         o   모듈 (Module): 총계정원장(General Ledger, GL)

&#x20;         o   전표번호(Batch no): 자동으로 전표 번호가 생성됩니다. (입력된 내용을 확인하기 위해 F3를 누르기)

&#x20;         o   전표일자(Date): 반드시 회계 월 이내

&#x20;         o   회계 월(Month): 장부기장 해당 월 선택 (예: 2015년 6월 기장: 201506)

&#x20;         o   적요(Description (VN)): 베트남어

&#x20;         o   적요(Description (EN)): 영어

&#x20;         o   적요(Description (KR)): 한국어

&#x20;         o   전표상태(Status):

&#x20;               §  Hold - 임시저장표

&#x20;               §  Released - 처리되어 장부(회계)에 반영된 전표

·         회계 처리

표시할 정보 항목 선택: 설명, 세금, 비용코드, 이익코드, 공정코드 등

&#x20;         o   **신규를** 선택하여 입력 창을 엽니다.

<figure><img src="../.gitbook/assets/image (88).png" alt=""><figcaption></figcaption></figure>

&#x20;         o   차변(Debit) 계정: 선택하거나 직접 입력하기 위해 F3키를 누르기

&#x20;         o   대변(Credit) 계정: 선택하거나 직접 입력하기 위해 F3키를 누르기

&#x20;         o   통화단위(Currency unit): 통화단위 선택(VND, USD, KOR 등)

&#x20;         o   환율(Rate Exchange): VND 이외의 기타 통화에 대한 환율 입력

&#x20;         o   금액(Cury Amount): 외화 금액 입력

&#x20;         o   환산 후 금액: 프로그램이 자동으로 계산하거나 사용자가 직접 입력 가능

&#x20;         o   적요 (VN), (EN), (KR): 업무 정보란의 설명 내용을 기준으로 자동 표시됨. 사용자는 직접 입력하여 세부 설명을 수정 가능

&#x20;         o   직원/ 공급업체/ 고객: 사용자가 직접 입력하거나 F3 키를 눌러 아래에 설정된 해당 코드 선택

&#x20;               §  직원 목록(현금 모듈),

&#x20;               §  공급업체 목록,

&#x20;               §  고객처 목록(외상매출금 모듈).

&#x20;         o   세금계산서 번호/ 세금계산서 일자/ 일련번호: 세금계산서 정보(있는 경우)를 입력하면 프로그램이 자동으로 부가가치세 매입/매출 명세서에 반영 가능

&#x20;         o   비용코드/ 매출코드/ 공정코드: 내부 보고나 원가 계산 목적으로 사용

**저장을** 선택하여 입력한 정보 저장

**삭제를** 선택하여 불필요하게 입력된 행 삭제

**복사를** 선택하여 이미 입력된 전표의 모든 정보 불러오기

**처리를** 선택하면 전표의 상태가 **Released로** 전환



#### 1.2.2   채권채무 상계

채권·채무 상계를 하고자 할 때 사용하는 기능입니다.

**신규를** 선택하여 입력 창을 엽니다.

분개 정보 부분과 입력 그리드 부분은 일반 분개 입력과 동일하게 진행됩니다.

&#x20;

**주의**: 거래처별 정확한 채권·채무 내역이 표시되도록 공급처/ 고객코드/ 직원 정보를 입력해야 합니다.
