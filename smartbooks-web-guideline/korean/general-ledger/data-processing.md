# 1.3.	데이터 처리

#### 1.3.1.  전기

연말에는 일반원장이 모든 거래를 다음 회계연도의 계정 잔액으로 마감합니다.

&#x20;

#### 1.3.2.  전표 마감 해제(Un-release Batch):

이 기능은 사용자가 전기(Post) 또는 확정(Release)된 전표를 원래 상태인 ‘보류(Hold)’ 상태로 복구할 수 있도록 도와줍니다.

<figure><img src="../.gitbook/assets/image (91).png" alt=""><figcaption></figcaption></figure>

상태 복원  안내

·         모듈: 마감 해제가 필요한 회계 모듈을 선택합니다.

&#x20;         o   일반원장(GL)

&#x20;         o   외상매입금(AP)

&#x20;         o   외상매출금(AR)

&#x20;         o   현금 회계(CA)

&#x20;         o   재고(IN)

&#x20;         o   고정자산(FA)

·         전표 조정 기간(Period): 전기된 월 또는 연 단위로 선택할 수 있으며, 불러오기(로드) 버튼을 누르면 마감된 분개 목록이 표시됩니다.

&#x20;

화면 하단에는 선택한 회계 기간에 발생한 마감된 전표들이 표시됩니다.

·         마감 해제가 필요한 전표를 선택합니다.

·         \[실행]을 눌러 작업을 진행합니다.

#### 1.3.3  월별 마감

회계 장부를 마감하여, 권한이 없는 사용자가 수정할 수 없는 기능입니다.
