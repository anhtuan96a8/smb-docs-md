# 4.1.	목록

#### 4.1.1.   부서 목록

회사의 조직도에 기초하여 분류된 각 부서를 등록 및 관리할 수 있습니다.

“신규”를 선택하여 새로운 “부서코드”와 “부서명(EN, VN)”을 입력하고, “저장”을 클릭합니다.

삭제할 항목을 선택한 후, “삭제”를 클릭합니다.

#### 4.1.2.   직급 목록

직급등록은 조직 내 직무의 종류, 책임 정도를 분류하는데 사용됩니다.

**“신규”** 버튼을 선택하여 새로운 “직급코드”와 “직급명(EN, VN)”을 입력하고, **“저장”** 버튼을 클릭합니다.

삭제할 항목을 선택한 후, “**삭제**”를 클릭합니다.

&#x20;

#### 4.1.3.   사원 목록

사원등록은 현금 또는 은행 계좌를 통하여 급여 이체시 사용됩니다.

사원 정보(사원코드, 사원명, 부서, 직책, 전화번호, 주소)를 입력하여 “저장” 버튼을 클릭합니다.

삭제할 항목을 선택한 후, **“삭제”**&#xB97C; 클릭합니다.
