# 12.4 Onhand Stock Setting

Onhand stock setting allow users to set the configugation and linked warehouse for each inventory document (eg. Purchase order, Stock issue, Stock transfer…).\
Thus, system determines which warehouse is used or created when it processes the vouchers.
