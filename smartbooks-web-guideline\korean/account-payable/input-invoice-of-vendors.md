# 2.2.	보고서 (Reports)

#### 2.2.1.   매입처별 전표 상세내역

·         시작 날짜와 종료 날짜를 선택합니다.

·         통화 유형을 선택합니다.

·         331 또는 338 계정을 선택합니다.

·         모든 매입처 또는 개별 매입처를 선택합니다.

보고서를 보려면 **보고서 인쇄**, 또는 **Excel 파일로** 내보내기 선택합니다.

<figure><img src="../.gitbook/assets/image (111).png" alt=""><figcaption></figcaption></figure>

모든 매입처에 대한 보고서를 엑셀 파일로 내보내는 경우

·         Sum 시트: 매입처별 왜상매입금 내역을 종합합니다. 매입처를 클릭하면 해당 업체의 상세 시트로 이동합니다.

·         상세 시트: DOCSMAP을 클릭하여 Sum시트로 돌아갑니다.

<figure><img src="../.gitbook/assets/image (112).png" alt=""><figcaption></figcaption></figure>
