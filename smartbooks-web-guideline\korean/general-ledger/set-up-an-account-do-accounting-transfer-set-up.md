# 1.1.	항목

### 1.        총계정원장의 체계 <a href="#toc212541644" id="toc212541644"></a>

총계정원장(GL)은 본 회계시스템의 핵심에 해당합니다.

사용자가 현금, 미수금, 미지급금, 고정자산에 입력하는 모든 거래는 총계정원장에 기록됩니다.

총계정원장은 설정, 입력, 처리, 분개장 및 보고서까지 5개 부문으로 구성됩니다.

#### 1.1.1.  계정 항목

스마트북은 2014년 12월 22일자 재무부가 공포한 시행규칙 200/TT-BTC에 규정된 주요 계정과목 공합니목록을 제공합니다.

기존에 규정된 계정과목 레벨 2과 레벨 3 등의 하위계정(최대 레벨5까지) 설정이 가능합니다.

<figure><img src="../.gitbook/assets/image (80).png" alt=""><figcaption></figcaption></figure>

계정 생성 방법

·         계정 번호: 하위계정 코드 입력. \* 참고: Smartbooks는 하위 계정코드는 7자리입니다.

·         계정과목(VN): 베트남어 적요 입력

·         계정과목(EN): 영어 적요 입력

·         계정과목(KR): 한국어 또는 기타 언어 적요 입력

·         계정분류: 자산, 부채, 매출, 비용 등의 성능에 맞는 계정 선택

·         계정그룹: 주요 계정 그룹 선택(계정코드의 앞 3자리)

·         상태: 계정과목의 사용 상태 설정: 사용(Active), 미사용(Inactive)

설정된 계정을 저장하기 위하여 저장(Save), 신규 계정과목 생성시 신규(New), 활성화된 창을 닫을 때에는 닫기(Close) 버튼 클릭

기존 계정과목 삭제 방법

·         삭제를 원하는 계정과목 선택

·         삭제(Delete) 버튼 클릭

참고: 계정과목 설정은 일반분개, 원장 및 재무제표에 직접적인 영향을 미칠 수 있으므로, 계정

삭제 전에는 신중히 고려해야 합니다. 이는 삭제된 계정과 관련된 거래가 존재할 경우 보고서의 정확성이 떨어질 수 있음을 의미합니다.

#### 1.1.2  계정 그룹 항목

Smartbooks는 시행규칙 200/2014/TT-BTC(발표일: 2014.12.22)에 규정된 레벨 1과 레벨 2의 계정과목 목록을 제공합니다.

사용자는 재무부로부터 승인받은 레벨 1 과 레벨 2 이외의 계정과목을 추가할 수 있습니다.

<figure><img src="../.gitbook/assets/image (81).png" alt=""><figcaption></figcaption></figure>

계정 그룹 생성:

·         그룹코드: 계정의 레벨 1 코드 형태

·         그룹명(VN): 베트남어 적요

·         그룹명(EN): 영어 적요

·         그룹명(KR): 한국어 적요

설정된 계정을 저장하기 위하여 **저장(Save)**, 신규 계정과목 생성시 **신규(New)**, 활성화된 창을 닫을 때에는 **닫기(Close)** 버튼 클릭

#### 1.1.3  대차대조표 설정

Smartbooks는 시행규칙 200/2014/TT-BTC(발표일: 2014.12.22)에 규정된 대차대조표 계정이 설정되어 있습니다.

사용자는 비활성화 체크를 통해 대차대조표 상 불필요한 사항을 삭제할 수 있습니다

<figure><img src="../.gitbook/assets/image (82).png" alt=""><figcaption></figcaption></figure>

·         1 단계: 코드, 적요(VN, EN, KR), 합계(합계 함수 사용), 상태 (대차대조표에 표시되거나 숨겨짐) 등의 정보설정

·         2 단계: 대차대조표 상 코드에 따라 잔액 지정.

&#x20;         o   구분(차, 대): 두 가지 유형 (D: 차변 잔액, C: 대변 잔액)

&#x20;         o   계정과목: F3키를 눌러 계정과목을 선택

&#x20;         o   잔액상태: 대차대조표 상에 계산되는 잔액은 음수 (-) 또는 양수 (+)로 반영

&#x20;         o   코드: 대차대조표에 사용되는 코드

&#x20;         o   기초잔액, 기말잔액: 마지막으로 열린 대차대조표 열기의 기초잔액, 기말잔액 (데이터 확인용)

<figure><img src="../.gitbook/assets/image (83).png" alt=""><figcaption></figcaption></figure>

·         3 단계: 상위 수준의 대차대조표 코드에 자회사 대차대조표 코드 설정

<figure><img src="../.gitbook/assets/image (84).png" alt=""><figcaption></figcaption></figure>
