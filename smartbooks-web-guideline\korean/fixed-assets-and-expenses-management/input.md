# 5.3.	데이터처리

#### 5.3.1.   감가상각비 계상

·         전표번호: 순번에 따라 자동 부여

·         작성일자: 회계처리일

·         회계월: 회계처리월을 기준

·         적요(VN/EN)

당기 감가상각비 계상할 고정자산 코드의 상세내역이 그리드에 표시되며, “저장”을 눌러 감가상각 분개를 저장합니다

#### 5.3.2.   선급비용 계상

기간별 자동 배분을 계산하려면 배분해야 할 선급비용을 선택하여 분개 입력 화면을 엽니다.

·         전표번호: 순번에 따라 자동 부여

·         전표작성일: 회계처리일

·         회계월: 회계처리 기준 월

·         적요 (VN/EN)

당기 선급비용 코드의 상세내역이 그리드에 표시되며, 저장 버튼을 눌러 배분 분개를 저장합니다
