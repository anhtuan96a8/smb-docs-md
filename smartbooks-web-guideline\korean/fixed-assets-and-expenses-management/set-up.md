# 5.2.	입력

#### 5.2.1.   고정자산 처분

“신규”를 클릭하여 입력 창을 엽니다.

<figure><img src="../.gitbook/assets/image (170).png" alt=""><figcaption></figcaption></figure>

**일반 정보**

·         전표 번호: 소프트웨어가 자동으로 업데이트합니다.

·         전표일자: 청산일자 입력

·         회계 월: 전표일자를 기준으로 소프트웨어가 자동 업데이트합니다.

·         고객 코드: F3 키를 눌러 고객 코드 선택

·         판매전표번호

·         처분: 고정자산 또는 선급비용

·         수익·비용 계정: 처분 시 발생한 이익 또는 손실을 인식할 계정을 F3으로 선택합니다.

·         통화 / 환율:

·         세율

·         적요 (VN/EN/KR)

<figure><img src="../.gitbook/assets/image (171).png" alt=""><figcaption></figcaption></figure>

**외상매출금전표 생성**을 선택하면, 사용자 입력을 위한 세금계산서 관련 정보 입력란이 표시됩니다.

**상세 정보:**

·         자산코드 / 자산명: 처분할 고정자산의 코드를 F3으로 선택합니다.

·         계정: 고정자산을 인식할 계정을 F3으로 선택합니다.

·         적요 (VN, EN, KR):

·         통화/ 환율:

·         수량/ 단위: 처분할 자산의 수량입

·         취득원가: 고정자산의 최초 취득가

·         감가상각 금액: 이미 감가상각된 금액

·         잔존가액: 남은 자산 금액

·         당월 감가상각 금액: 이번 달 감가상각 금액

한가지 자산코드로 여러 차례 처분가능합니다.

#### 5.2.2.   고정자산 재평가

고정자산 처분 단계와 동일하게 수행합니다.
