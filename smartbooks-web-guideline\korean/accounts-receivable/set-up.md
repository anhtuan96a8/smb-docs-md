# 3.1.	외상매출금 입력

외상매출금 두 가지 주요 기능이 있습니다.

·         입력 (Input)

·         보고서 (Reports)

<figure><img src="../.gitbook/assets/image (119).png" alt=""><figcaption></figcaption></figure>

### 3.1.       외상매출금 입력

재고이동과 관련된 영수증 입력의 경우, 재고관리 모듈의 판매 항목에서 입력합니다.

본 모듈에는 서비스에 대한 매출 세금계산서 입력을 수행합니다. (재고와 관련 없는 경우)

&#x20;

#### 3.1.1.   매출 세금계산서 입력

“신규”를 선택하여 입력 화면을 엽니다. 모든 전표는 처리되기 전까지 기본적으로 보류(Hold) 상태로 입력됩니다.

<figure><img src="../.gitbook/assets/image (120).png" alt=""><figcaption></figcaption></figure>

·         일반내역(General area):

&#x20;          o   모듈전표번호(Voucher No.): 소프트웨어가 순차적으로 자동 기입

&#x20;          o   전표일자(Voucher Date): 영수증 날짜는 회계기간이내여야 합니다.

&#x20;          o   회계 월(Month): 전표의 월 기준으로 자동 기입

&#x20;          o   세금계산서 번호/ 시리얼 번호/ 세금계산서 발행일: 세금계산서에 표시된 정보를 입력합니다.

&#x20;          o   매출처: F3을 눌러 매출처 코드를 선택합니다.

&#x20;          o   외상매출금 계정과목: F3키를 눌러 목록에서 선택

&#x20;          o   통화 유형/환율: VND 기준 환율로 지급할 통화를 선택합니다.

&#x20;          o   지급 방식: F3키를 눌러 목록에서 선택

&#x20;          o   적요(Description): 해당 거래 내용을 입력(VN-EN)

·         발주서 내역

&#x20;          o   발주서 번호

&#x20;          o   선하증권 번호

&#x20;          o   배송지/수령지

&#x20;          o   계약 번호

&#x20;          o   컨테이너 번호

&#x20;          o   계약일

&#x20;          o   구매자/운송업체

&#x20;          o   비고

<figure><img src="../.gitbook/assets/image (121).png" alt=""><figcaption></figcaption></figure>

·         상세내역(Detail area): 영수증의 세부 내역을 나타냅니다. 아래의 데이터 시트 부분에 입력이 가능합니다.

&#x20;         o   계정과목(Account): F3키를 눌러 해당 매출계정을 선택

&#x20;         o   통화 유형/환율: 통화 선택 (소프트웨어는 일반내역에서 기본값으로 기입합니다).

&#x20;         o   품목 ID: F3을 눌러 출하한 상품(있는 경우) 선택

&#x20;         o   통화 유형: 일반내역에서 기본값을 자동으로 기입합니다.

&#x20;         o   수량

&#x20;         o   단가/환산 단가: 단가를 입력하고, 환산 단가는 입력된 환율에 따라 자동으로 기입됩니다.

&#x20;         o   금액/환산 금액: 금액을 입력하고, 환산 금액은 입력된 환율에 따라 자동으로 기입됩니다.

&#x20;         o   세율.

&#x20;         o   세금/환산 세금 금액

&#x20;         o   공정 코드/비용 코드/이익 코드

&#x20;         o   적요: 적요 입력 (소프트웨어는 일반내역에서 기본값으로 기입합니다).

#### 3.1.2.   모듈전표 마감

본 항목은 외상매출금 모듈에 입력된 전표를 총계정원장에 전기하는 기능을 수행합니다. 전기할 항목 모두 또는 월별로 선택하여 “실행”버튼을 누릅니다.

·         모듈전표를 개별적으로 마감합니다

&#x20;         o   전표 목록에서 마감할 모듈전표를 선택합니다.

&#x20;         o   “마감”를 선택합니다.

·         모듈전표를 일괄적으로 마감합니다.

&#x20;         o   목록에서 마감되지 않은 전표(보류 상태)를 필터링합니다.

&#x20;         o   마감할 전표를 선택합니다.

&#x20;         o   “마감 완료”를 선택합니다.
