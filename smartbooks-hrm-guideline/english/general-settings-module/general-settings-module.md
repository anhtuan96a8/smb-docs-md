# GENERAL SETTINGS MODULE

The GENERAL SETTINGS module (Figure IV.1) use to establish initial information or changing existing information and application for all the company. This module includes the following items:

* Company information.
* List of departments, divisions, teams, groups, positions ...
* List of gender, ethnicity, religion, education level ...
* Registration a new user or delete an existing user.
* Decentralized use
* …
