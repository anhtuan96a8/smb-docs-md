# The Create/ Adjust Hazards Item

## **Item description**

Each employee will have a hazards code corresponding to the job for enterprise classify and provide the suitable regime.

## **Implementation Instruction**

### **Guide to create new**

In the Taskbar, select item ![](<../.gitbook/assets/0 (65).png>).

Create new Hazards code according to the instructions in section **II.2**. Once created successfully, the data will be displayed as Figure IV.11.1.

![IV.11.1](<../.gitbook/assets/1 (52).png>)

**Explanation**

* Hazards percentage: The enterprise assesses the hazardous level to calculate allowances for employees based on this ratio. For example: The hazards percentage is 5 - that means, the employee's hazards allowance will be equal to 5% of the salary under to the company regulations.

### **Implementation to edit, delete and export data**

To edit, delete and export data to Excel file, follow the instructions in the section **II.3, II.4, II.5, II.6.**
