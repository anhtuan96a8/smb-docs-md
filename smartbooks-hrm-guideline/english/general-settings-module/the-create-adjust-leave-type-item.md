# The Create/ Adjust leave type Item

## **Item description**

This Item uses to register the leave for employees: annual leave, sick leave, and maternity …

## **Implementation Instruction**

### **Guide to create new data**

In the Taskbar, select item![](<../.gitbook/assets/0 (67).png>).

Create the new leave types according to the instructions in section **II.2**. Once created successfully, the data will be displayed as Figure IV.8.1.

![IV.8.1](<../.gitbook/assets/1 (29).png>)

**Explanation:**

* Leave type: Enter code of leave type.
* Salary; Without salary; Insurance; annual leave; Miscarriage; Antenatal check-up ...: tick the information depends on the type of leave.
* No approvement: Applies to the absent case without superior approval
* Number of months, days: The maximum period of leave is allowed when registering the respective leave type. For example, maternity leave is only approved 06 months maximum.

### **Implementation to edit, delete and export data**

To edit, delete and export data to Excel file, follow the instructions in the section **II.3, II.4, II.5, II.6.**
