# Setting the workmanship Group, Level

## **Function description**

This function is used to manage the workmanship level, each skill level corresponds to the group of different skill levels and has a different allowance according to the company's regulations. Users can fill information of workmanship group, levels directly into the toolbar or get data from the excel file.
