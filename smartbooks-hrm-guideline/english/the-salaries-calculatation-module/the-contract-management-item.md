# The Contract Management Item

## **Item description**

This Item manages the contract and the labor contract process of employees from the first probationary contract until the permanent full-time labor contracts and contract appendices.

## **Implementation instruction**

### **Guide to creating new**

In the Taskbar, select ![](<../.gitbook/assets/0 (137).png>) .

To create information, follow the instructions in section **II.2**. The data will be displayed as shown in Figure VII.8.1.

![VII.8.1](<../.gitbook/assets/1 (142).png>)

**Interface explanation**

* Main Contract: showing the latest contract code that the software automatically associates with the staff code in case of creating contract appendix.
* **Noted**: In the case of a permanent full- time labor contract, there is no need to select the End Date.

### **Instructions to edit, delete and export data**

To edit, delete and export data to Excel file, follow the instructions in section **II.3, II.4, II.5, II.6.**

### **Reports explanation**

* List of contracts by signing date: the contracts list of employees is working in the selected period.
* List of contracts by expiration date: the contracts list will expire during the period of employees who are working.
* List of contracts is affecting: the contract list is doing in force in the selected period.
* List of the expected signing contracts: the contract list will be sign expectedly in the selected period
* Print probation contract: This function to print the probation contract.
* Print Official contract: This function to print the Official contract.
