# 엑셀 파일로 데이터 내보내기

&#x20;기능 설명

목록에 표시된 데이터를 엑셀파일로 저장할 수 있습니다.

*
  1. &#x20;실행 안내

&#x20;(1) 데이터를 내보내기 할 열을 선택합니다-> (2) ![](<../.gitbook/assets/0 (105).png>)을 클릭합니다 -> (3) 파일을 저장하고 필요한 경우 파일 이름을 변경할 폴더를 선택합니다 -> (4) **“Save”**&#xC744; 클릭하여 데이터 출력을 엑셀 파일로 내보냅니다(그림 II.5.1)

![II.5.1](<../.gitbook/assets/1 (45).png>)

참고:

* 상단의 **“Choosing all”** 버튼을 클릭하여 모든 행을 선택하거나 재 클릭하여 모든 선택을 해지합니다.
* Excel 파일 손상 사례:

\+ 성조 또는 특수 문자를 사용하여 베트남어 이름 폴더에 파일을 저장합니다.

\+ 성조 또는 특수 문자를 사용하여 베트남어로 파일 이름을 설정합니다.
