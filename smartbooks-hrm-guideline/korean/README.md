# I.	용어 안내

## 섹션 내용

인적 자원 관리 과정에서 여러 개별 단어가 사용됩니다. 본 섹션에서는 이에 대해 설명합니다.

## 설명:

* 기능 상자 (그림1.2.1)은 다음과 같습니다:

\+ 보고서 목록: 신입사원 목록; 시간 기록 보고; 월간 급여보고서

\+ 실행 목록: 시간 관리 계산, 급여 계산, 시간 관리 데이터 불러오기...

![I.2.1](<.gitbook/assets/0 (99).png>)

데이터 그리드: 그림1.2.2

*
  * 소프트웨어를 입력한 후 모든 정보의 데이터를 표시합니다.
  * 사용자는 그리드에서 데이터를 편집할 수 있습니다.
  * 데이터 필터링 기능은 ![](<.gitbook/assets/1 (78).png>)아이콘의 오른편에 위치해 있습니다.

![I.2.2](<.gitbook/assets/2 (25).png>)

도구 모음(그림I.2.3): 정보를 입력하고 데이터를 검색합니다.

![I.2.3](<.gitbook/assets/3 (24).png>)

작업 표시줄 (그림 1.2.4): 화면 왼쪽의 인터페이스에는 소프트웨어의 4가지 주요 모듈과 각 모들 내의 항목 및 관리 기능 목록이 포함되어 있습니다.

| 모듈   | 항목 및 기능                                  |
| ---- | ---------------------------------------- |
| 일반설정 | 회사 정보; 신규 사용자 등록 또는 현재 사용자 삭제; 사용자 권한... |
| 직원정보 | 직원 정보; 가족 정보; 부서 이동….                    |
| 근태관리 | 초과근무 등록, 연차 등록, 근무시간 산출…                 |
| 급여   | 급여 순위 설정; 부양 가족 목록; 급여명세서….              |
