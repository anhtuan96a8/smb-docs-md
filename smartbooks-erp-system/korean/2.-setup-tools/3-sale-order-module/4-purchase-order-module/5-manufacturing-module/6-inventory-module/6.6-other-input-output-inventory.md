# 6.6 기타 입출고 (Other Input/Output Inventory)

기타 입출고는 자재 반납, 외주 가공을 위한 원자재 출고 등과 같은 기타 사유로 인해\
재고를 증감시킬 때 사용됩니다.

#### 6.6.1. 기타 입출고 내역 조회

**메뉴 → 재고 관리 → 기타 입출고(Other Input/Output Inventory)** 로 이동하여 전표 목록을 확인합니다.

**\[필터(Filter)]** 버튼을 선택하거나, 각 필드 상단의 필터 박스에 조건을 입력하여 조회할 수 있습니다.

* **From date / To date**: 조회 기간 설정

<figure><img src="../../../../../.gitbook/assets/image (33).png" alt=""><figcaption></figcaption></figure>

#### 6.6.2 기타 입출고 생성

**\[신규]** 버튼을 선택하여 입력 화면을 엽니다

<figure><img src="../../../../../.gitbook/assets/image (34).png" alt=""><figcaption></figcaption></figure>

**일반 정보**

* **RefNbr**.: 자동 생성되는 시스템 저장 번호
* **일자 (Date)**
* **Reference No**.: 자동 생성
* **창고 코드 (Warehouse code)**: F3 키를 눌러 드롭다운 목록에서 선택하거나 직접 입력
* **통화 (Currency ID)**: 거래 통화 (기본값은 VND)
* **입고 유형 (Receipt type)**
* **거래처 코드 (Vendor ID)**
* **설명 (베트남어)**

**상세 정보**\
&#xNAN;**\[신규]** 버튼을 선택하여 품목을 추가합니다.

<figure><img src="../../../../../.gitbook/assets/image (35).png" alt=""><figcaption></figcaption></figure>

#### 6.6.3  기타 입출고 내역 불러오기

품목 정보를 담은 Excel 파일이 있는 경우, 이를 불러와 시간을 절약할 수 있습니다.

* **\[템플릿 선택(Select Template)]**: 올바른 형식의 템플릿 파일을 다운로드합니다. 안내에 따라 각 열에 데이터를 입력합니다.
* **\[불러오기 선택(Select Import)]**: 불러오기 창을 엽니다.
* **\[파일 선택(Choose)]** 버튼 클릭 → 파일 위치로 이동 → 파일 선택 → **\[열기(Open)] → \[업로드(Upload)]**
* 오류 메시지가 표시되는 경우(있는 경우), 해당 오류를 확인하고 수정합니다.
* **\[파일 업로드 / 불러오기(Upload file / Import)]** 실행

<figure><img src="../../../../../.gitbook/assets/image (36).png" alt=""><figcaption></figcaption></figure>

#### 6.6.4  기타 입출고 수정

기타 입출고 전표가 **보류(Hold)** 상태일 경우, 해당 정보를 수정할 수 있습니다.\
**처리 완료(Posted)** 상태일 경우에는 데이터가 잠기며 수정이 불가능합니다.

기타 입출고 전표를 수정하려면 **수정 아이콘** 또는 **전표 번호**를 선택하여 상세 화면을 엽니다.\
새 전표 생성 방식과 유사하게 **일반 정보 및 상세 정보**를 수정한 후 **\[저장(Save)]** 합니다

<figure><img src="../../../../../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure>

**재고 조정 삭제 / 출력 / 엑셀 내보내기**

* 전표 또는 품목을 삭제하려면 **삭제 아이콘**을 선택합니다.
* 여러 전표 또는 여러 품목을 삭제하려면 각 행의 **왼쪽 체크박스를** 선택한 후 **\[삭제(Delete)]** 를 클릭합니다.
* 전표를 출력하려면 각 행의 **출력 아이콘(프린트 아이콘)** 을 선택합니다.
* 엑셀 파일로 내보내려면 각 행의 **왼쪽 체크박스**를 선택한 후 **\[내보내기(Export)]** 를 클릭합니다.
