# 6.3 재고 이동(Stock Transfer)

재고 이동은 하나의 창고에서 다른 창고로 품목을 옮겨야 할 때 사용됩니다.\
예: 원자재 창고에서 공장으로, 또는 생산 라인에서 완제품 창고로 이동하는 경우

#### 6.3.1 재고 이동 생성

**메뉴 → 재고 관리 → 재고 이동**으로 이동하여 이동 목록을 엽니다.

<figure><img src="../../../../../.gitbook/assets/image (28).png" alt=""><figcaption></figcaption></figure>

조회 기간(From date / To date)을 변경하여 이전의 이동 내역을 확인할 수 있습니다.\
새로운 재고 이동을 생성하려면 **\[신규]** 버튼을 선택하여 입력 화면을 엽니다.

<figure><img src="../../../../../.gitbook/assets/image (29).png" alt=""><figcaption></figcaption></figure>

필수 정보( \* 표시된 항목 )를 입력합니다.\
재고 이동 요청서에 정보가 입력되어 있는 경우, 해당 품목이 자동으로 불러와집니다. 사용자는 실제 재고 수량과 일치하도록 수량을 확인하고 조정해야 합니다.

품목 상세 정보를 추가하려면 **\[신규]** 버튼을 클릭하여 필요한 정보를 입력합니다.\
품목에 LOT 번호가 있는 경우, 저장하기 전에 반드시 LOT 번호를 입력해야 합니다



#### 6.3.2 재고 이동 수정

이동 상태가 보류(Hold)일 경우 정보 수정이 가능합니다.\
이동이 **처리됨(Posted)** 상태일 경우에는 데이터가 잠기며 수정이 불가능합니다.

재고 이동 정보를 수정하려면 **수정 아이콘** 또는 **이동 번호**를 선택하여 상세 화면을 엽니다.\
일반 정보 및 상세 정보를 새 이동 생성 방식과 유사하게 수정한 후 **저장(Save)** 합니다.

**재고 이동 삭제 / 출력**

* 재고 이동 또는 품목을 삭제하려면 **삭제 아이콘**을 클릭합니다.
* 여러 이동 또는 여러 품목을 삭제하려면 각 행의 **왼쪽 체크박스**를 선택한 후 **삭제(Delete)** 를 클릭합니다.
* 재고 이동서를 출력하려면 각 행의 **출력 아이콘(Print)** 을 선택합니다.
