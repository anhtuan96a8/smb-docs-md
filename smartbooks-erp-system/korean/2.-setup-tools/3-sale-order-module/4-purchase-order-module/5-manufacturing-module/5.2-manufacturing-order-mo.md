# 5.2 	제조 지시서(MO)

이 모듈은 **제조 지시서**를 생성하는 데 사용됩니다. 지시서가 생성되면 원자재 출고 전표와 완제품 입고 전표가 자동으로 연결됩니다. 완제품 입고 수량이 MO 수량과 동일해지면 해당 주문이 완료됩니다.

#### 5.2.1     MO 전표 검색

**메뉴 -> 생산 -> 제조 지시서**를 선택하여 MO 화면을 엽니다.

<figure><img src="../../../../.gitbook/assets/image (16).png" alt=""><figcaption></figcaption></figure>

필터 버튼을 선택하거나 각 항목의 필터 박스에 조건을 입력하세요.

* **From date / To date**: 검토 기간
* **MO Status:** 제조지시서 상태

<figure><img src="../../../../.gitbook/assets/image (17).png" alt=""><figcaption></figcaption></figure>

#### 5.2.2     제조 지시서 생성

**제조 지시서**를 생성하는 경우 **‘신규’** 버튼을 클릭하여 입력 화면을 엽니다.

<figure><img src="../../../../.gitbook/assets/image (18).png" alt=""><figcaption></figcaption></figure>

**일반 정보**

* **MO No.**: 자동 생성
* **SO No.**: F3 키를 눌러 드롭다운 목록에서 관련된 SO(판매오더)를 선택
* **Date**: 제조지시서(MO) 작성일
* **BOM code**: 작성된 BOM 코드
* **BOM version**: 기본 버전이 자동으로 불러와짐. F3 키로 드롭다운 목록에서 선택 가능
* **MO Quantity:** 완제품 수량, SO에서 자동 불러오기되며 수정 가능
* **Warehouse code**: F3 키로 드롭다운 목록에서 창고 선택
* **Plant code**: F3 키로 드롭다운 목록에서 공장 선택
* **Machine Code**: 기계 코드
* **Molding Code**: 금형 코드
* **SO No**: 판매 오더 번호
* **Started date**: 제조 시작일
* **Ended date**: 제조 종료일
* **PIC**: 담당자
* **Note:** 비고
* **Description VN/EN/KR**: 베트남어/영어/한국어 설명

**세부 정보**\
**구성품 (Component)**: 제조지시서에서 사용하는 원자재

* BOM에서 자동 불러오기 (품목 코드, 품명, 단위, 기준 수량)
* **Planned Quantity**: 제조 수량 기준 자동 계산
* **Actual Quantity**: 제조 수량 기준으로 임시 자동 계산되며, 출고요청서 기준으로 추후 변경 가능
* **Variance Quantity**: 계획 수량과 실제 수량의 차이
* **Total stock / Stock on line**: 현재 재고 및 생산라인 재고 수량
* **Note**: 비고

**공정 (Operation, 해당 시)**

* BOM에서 자동 불러오기 (공정 코드, 공정 명, 표준 소요 시간)
* **Actual Duration**: 실제 소요 시간
* **Variance Duration**: 계획 시간과 실제 시간의 차이

**입출고 목록 (Receipt/ Issue List)**

* 해당 제조지시서에 사용된 재고 요청서와 완제품 입고서가 자동 불러오기됨

**소형 BOM (Small BOM)**

* 생산 구간이 나뉘어진 BOM의 경우, 대형 완제품에 대해 생산지시서를 생성하면 시스템이 자동으로 관련 소형 완제품에 대한 제조지시서를 추가로 생성함. 이 하위 제조지시서는 소형 MO 목록에 표시됨.
