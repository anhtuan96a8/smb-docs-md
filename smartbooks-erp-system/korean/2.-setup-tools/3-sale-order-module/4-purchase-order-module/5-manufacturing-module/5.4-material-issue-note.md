# 5.4.	자재 출고 전표

이 기능은 생산을 위한 자재 출고 전표를 생성하는 데 사용됩니다.\
해당 전표는 제조오더(MO)와 연동되어, MO 상태를 **Open(개시)** 에서 **Processing(진행 중) 또는 Completed(완료)** 로 업데이트합니다.

경로: **메뉴(Menu) → Manufacturing(생산관리) → Material Issue Note(자재 출고 전표)** 를 선택하여 전표 목록을 엽니다.

**Filter 버튼을 선택하거나 각 필드의 필터 박스에 조건을 입력**하여 원하는 자료를 조회합니다.

* From date / To date: 검토 기간 (조회 기간)

#### 5.4.1     자재 출고 전표 생성

자재 출고 전표를 생성하려면, \[new] 버튼을 선택하여 입력 화면을 엽니다.

<figure><img src="../../../../.gitbook/assets/image (22).png" alt=""><figcaption></figcaption></figure>

**일반 정보**

* **RefNbr**.: 자동 생성됨
* **Date**: 날짜
* **Warehouse Code**: 입고 창고, F3을 눌러 드롭다운 리스트에서 선택
* **Production Line**: 생산 라인
* **MO No**.: F3을 눌러 드롭다운 리스트에서 제조오더(MO) 선택 → 수량 자동 불러오기
* **Note**: 비고

**상세 정보**

* BOM에서 자동으로 불러옴 (품목 코드, 품목명, 단위, 창고 코드, 완제품 코드, 입고 전표, MO 번호)
* **Business Quantity**: 출고 수량 → 편집 아이콘을 눌러 수량 입력
* **Note**: 비고
* 품목을 추가하려면 **\[New] 버튼**을 눌러 입력 창을 열기
* 저장

#### 5.4.2. 자재 출고 전표 수정

\
자재 출고 전표는 **Hold 상태일 때만 수정 또는 삭제 가능합**니다. Released 상태가 되면 회계 모듈과 연동되므로 데이터가 잠깁니다.

* 자재 출고 전표를 수정하려면, **수정 아이콘 또는 전표 번**호를 클릭하여 상세 화면을 열고, 일반 정보 또는 상세 정보를 직접 변경 → 저장
* 전표를 삭제하려면 **삭제 아이콘 클릭 후 안내에 따름**
* 여러 전표를 한 번에 삭제하려면 **왼쪽 체크박스를 선택** → \[Delete]
