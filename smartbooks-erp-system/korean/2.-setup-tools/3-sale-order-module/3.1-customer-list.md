# 3.1 	고객 목록

이 모듈은 S\&S ERP 시스템에서 주문서를 작성하기 위한 기본 프로세스를 제공합니다. 판매 주문 프로세스는 다음 단계가 포함됩니다.

·       가격표/견적서 **작성**: 구매가 필요한 고객에게 알리고 보냅니다.

·       판매 주문서 **작성**

·  **판매 단가 업데이트**

·       판매 출고서 **작성**: 상품이 출고 준비 완료된 경우

·       반품 전표 **작성**(있는 경우)

·       세금계산서 번호 **업데이트**

### 3.1        고객 목록 <a href="#toc201592035" id="toc201592035"></a>

#### 3.1.1     신규 고객 생성 <a href="#toc201592036" id="toc201592036"></a>

새 고객을 생성하는 경우 메뉴 -> 목록 -> 고객 목록으로 이동합니다.

<figure><img src="../../.gitbook/assets/image (85).png" alt=""><figcaption></figcaption></figure>

고객 목록 화면은 그리드 테이블으로 표시됩니다. 목록에서 고객을 찾으려면 각 열의 필터 셀에 검색 조건을 입력하세요.

<figure><img src="../../.gitbook/assets/image (86).png" alt=""><figcaption></figcaption></figure>

**신규**를 선택하여 신규 고객 정보를 입력하는 창을 엽니다.

_**종합정보**_

·    **고객 코드**: 1개의 코드는 각 고객에 대해서만 고유할 수 있습니다.

·       **고객 이름**: 고객 이름(기업 이름 또는 개인 구매자 이름)

·       **거래 통화**: 거래에 사용되는 통화입니다. 고객이 2가지 이상의 통화를 사용하는 경우 해당 통화로 2개의 고객 코드를 작성해야 합니다. (예: Elensys\_VND, Elensys\_USD)

·       **활동**: 고객이 현재 회사와 거래를 지속하고 있습니다.

·       **판매 주문서**: 고객의 판매 주문 내역(주문 수, 총 금액, 세금, 할인 등)

·       **출고서**: 해당 고객 코드에 대한 배송 내역

<figure><img src="../../.gitbook/assets/image (87).png" alt=""><figcaption></figcaption></figure>

_**세부 정보**_

·       **탭 1. 고객 정보**

&#x20;        o   대표자: 대표이사, 관리자 또는 연락 담당자

&#x20;        o   주소: 해당 회사 주소

&#x20;        o   전화 번호: 회사 전화 번호(있는 경우)

&#x20;        o   이메일: 회사 또는 담당자의 이메일(있는 경우)

&#x20;        o   팩스: 팩스 번호(있는 경우)

&#x20;        o   세금 코드: 세금 코드 또는 회사 코드(있는 경우)

·      **탭 2. 기본 설정**

&#x20;        o   지급 기간 또는 방법: 지급 방법(현금/계좌이체/기간)

&#x20;        o   세율: 세율 유형(있는 경우)

&#x20;        o   미수금 계정(있는 경우)

&#x20;        o   매출 계정: 매출 계정(있는 경우)

&#x20;        o   선급 계정: 선급 계정(해당되는 경우)   &#x20;

·       **탭 3. 기타 정보**

&#x20;        o   고객의 은행 계좌 정보

·       **탭 4. 배송 주소:**

"새로 추가" 버튼을 눌러 추가 배송 주소를 생성합니다.

&#x20;        o   배송 주소

&#x20;        o   수취인

&#x20;        o   전화번호

&#x20;        o   이메일 주소

&#x20;        o   정보 1

&#x20;        o   정보 2

&#x20;        o   기본 설정: 일반 배송 주소인 경우 기본 설정 선택.

·       **메모 탭**

#### 3.1.2     동시에 여러 고객 입력 방법 <a href="#toc191471941" id="toc191471941"></a>

사용자가 여러 고객 목록을 보유하고 있는 경우, 엑셀 파일을 이용한 입력 방법을 선택할 수 있습니다.

\-        샘플 파일 **선택**: 엑셀 샘플 파일을 다운로드합니다. 제공된 가이드에 따라 데이터를 해당 컬럼에 복사합니다.

\-         데이터 **입력 선택**: 파일을 시스템에 업로드합니다.

&#x20;          o   파일이 저장된 폴더로 이동 -> import 파일 선택 -> Open 버튼 클릭.

&#x20;          o   일반적으로 엑셀 파일은 자동으로 읽히고, 그렇지 않는 경우 파일 이름을 클릭하세요.

&#x20;          o   표시된 오류 목록을 추적하고 정보를 수정하세요.

#### 3.1.3     고객 정보 수정

고객이 기업 정보를 수정하는 경우, 사용자는 ERP 시스템에서 정보를 수정할 수 있습니다.

새 고객을 생성하는 방식과 같습니다: 메뉴 -> 카테고리 -> 고객 카테고리로 이동합니다.

\-        고객 목록이 표시되면 수정 아이콘 또는 고객 코드를 선택하여 세부 정보 창을 엽니다.

\-        수정할 정보를 재입력한 후 저장 버튼을 클릭합니다

&#x20;         (참고: 고객 코드는 변경되지 않습니다)
