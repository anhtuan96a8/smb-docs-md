# 3.8 	매출 세금계산서 업데이트

주문이 고객에게 배송될 때 매출 세금계산서는 일련의 판매서류를 완전하고 채권을 기록하기 위해 실제 판매 주문에 해당하여 작성됩니다. 본 섹션은 경리가 발행할 세금계산서를 선택하는 데 도움이 됩니다.

<figure><img src="../../.gitbook/assets/image (115).png" alt=""><figcaption></figcaption></figure>

#### 3.8.1     세금계산서 번호 업데이트 <a href="#toc201592061" id="toc201592061"></a>

메뉴 -> 판매 -> **매출 세금계산서 업데이트**를 선택하여 세금계산서가 발행되지 않은 출고전표 목록을 엽니다. 전표 목록은 사용자가 열의 조건을 쉽게 필터링할 수 있도록 그리드 테이블 형식입니다.

시간 디스플레이 창을 변경하는 경우 From Date/To Date(날짜부터/날짜까지) 수정 -> **보기**를 선택합니다.

<figure><img src="../../.gitbook/assets/image (116).png" alt=""><figcaption></figcaption></figure>

매출 세금계산서 업데이트:

\-        발행하지 않는 세금계산서를 확인합니다.

\-        전표를 선택하려면, 해당하는 각 줄의 왼쪽에 있는 사각형 선택 -> **매출 세금계산서 업데이트합니다.**

<figure><img src="../../.gitbook/assets/image (117).png" alt=""><figcaption></figcaption></figure>

_종합정보_

\-        고객 코드: 자동 업로드

\-        고객 이름: 자동 업로드

\-        고객 이름(EN): 자동 업로드

\-        일련 번호: 송장 일련 번호

\-        인보이스 날짜

\-        세금계산서 번호

\-        총액
