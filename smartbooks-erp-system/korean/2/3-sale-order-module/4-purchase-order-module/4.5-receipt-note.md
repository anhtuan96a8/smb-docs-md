# 4.5.	구매 입고

입고 전표는 완제품/상품이 창고에 입고 준비가 되었을 때 비고를 작성하는 데 사용됩니다. 비고가 작성되면 관련된 주문서의 상태가 열기에서 처리 중 또는 완료로 변경됩니다.

#### 4.5.1     입고 전표 생성

**메뉴 → 구매 주문 → 입고 전표로** 이동하면 목록이 그리드 형식의 표로 표시됩니다.

**\[필터] 버튼을** 선택하여 조건 입력 화면을 열거나, 각 필드의 필터 상자에 조건을 직접 입력할 수 있습니다.

<figure><img src="../../../.gitbook/assets/image (57).png" alt=""><figcaption></figcaption></figure>

**필터 정보**\
• **사용 여부(Activated)**: 단가표의 상태 (활성 또는 비활성)\
• **단가표 번호(Price List No.)**: 단가표 번호 (기억하는 경우 입력)\
• **공급업체 코드(Vendor Code)**: F3 키를 눌러 드롭다운 목록에서 선택하거나 코드를 직접 입력\
• **품목 코드(Item Code)**: 해당 품목을 판매하는 모든 공급업체가 표시됨\
• **적용일(Validated Date)**: 단가표가 적용되는 시작 날짜\
• **만료일(Expired Date)**: 단가표가 비활성화되는 날짜

**입고 전표(Receipt Note)를 생성하려면, \[신규]** 버튼을 클릭하여 입력 화면을 엽니다.

<figure><img src="../../../.gitbook/assets/image (58).png" alt=""><figcaption></figcaption></figure>

_**기본 정보:**_

* **참조 번호(Reference No.):** 자동 생성되며, 수정할 수 없습니다.
* **주문일자(PO Date)**: 생성된 날짜
* **공급업체 코드(Vendor Code)**: F3 키를 눌러 드롭다운 목록에서 선택하거나 코드를 직접 입력
* **통화 코드(Currency Code)**: 이 거래에 적용되는 통화 (기본값은 VND). 다른 통화를 선택할 경우, 환율을 입력해야 합니다.
* **구매 주문 번호(PO No.)**: F3 키를 눌러 드롭다운 목록에서 구매 주문 선택
* **창고(Warehouse)**: F3 키를 눌러 창고 선택
* **통화 ID(Currency ID)**: 이 거래에 적용되는 통화 (기본값은 VND). 다른 통화를 선택할 경우, 환율을 입력해야 함
* **비고(Note)**
* **설명 (베트남어/영어/한국어)**

**세부 정보**\
구매 주문 번호(PO 번호)를 선택하면, 시스템에서 해당 주문에 포함된 품목들이 표시되며 사용자는 입고할 품목 코드를 선택해야 합니다. 이후 실제 입고 수량에 맞게 수량을 조정합니다.\
PO 번호 없이 직접 품목을 입력하려는 경우에는 **\[신규]** 버튼을 클릭하여 상세 입력 창을 엽니다.

* **품목 코드(Item code)**: F3 키를 눌러 드롭다운 목록에서 선택하거나 코드를 직접 입력
* **공급업체 코드(Vendor code)**: F3 키를 눌러 드롭다운 목록에서 선택하거나 코드를 직접 입력
* **창고(Warehouse)**: 재고를 입고할 창고입니다. 품목 목록에서 자동으로 불러오며, 변경하려면 F3 키를 눌러 드롭다운 목록에서 선택하거나 코드를 입력
* **단위(UOM)**: 자동으로 불러옵니다
* **세금 코드(Tax code)**: 품목 목록에서 자동 불러오기 되며, 원하는 VAT 코드로 수정 가능
* **할인율(% Discount)**: 해당 품목에 대한 할인 비율. 할인 금액은 자동으로 계산됨
* **금액(Amount)**: 세금 제외 금액, 자동 계산
* **세액(Tax Amount)**: 자동 계산
* **총액(Total Amount)**: 세금 포함 총액

#### 4.5.2     구매 입고 전표 수정

구매 입고 전표의 세부 사항은 **초안(Draft)** 상태일 때 수정할 수 있습니다.

다른 상태에 있을 경우, 해당 데이터는 다른 모듈과 연결되어 잠겨 수정할 수 없습니다.

\-        전표 수정하려면 수정 아이콘 또는 전표 번호를 클릭합니다

\-        필수 정보를 수정합니다. 저장.

#### 4.5.3    구매 입고 전표 삭제/인쇄

\-         구매 입고 전표를 삭제하려면 삭제 아이콘을 클릭하고 지침을 따르세요.

\-         여러 구매 입고 전표를 동시에 삭제하려면 삭제할 전표를 선택하고 **삭제** 버튼을 클릭합니다.

\-         구매 입고 전표를 인쇄하려면 인쇄 아이콘을 클릭합니다.

\-         엑셀 파일로 출력하려면 삭제할 전표를 선택하고 **데이터 출력 버**튼을 클릭합니다.
