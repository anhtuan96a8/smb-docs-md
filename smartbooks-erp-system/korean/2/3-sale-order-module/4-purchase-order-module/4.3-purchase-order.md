# 4.3	구매 주문서

구매 주문(Purchase Order)은 공급업체에 요청한 주문을 기록하는 데 사용됩니다.\
해당 공급업체의 단가가 **공급업체 단가표**에 미리 입력되어 있다면, 구매 주문과 연동되어 단가가 자동으로 불러와집니다. 또한, 단가를 수동으로 조정하거나 할인 금액(있는 경우)을 추가할 수도 있습니다.

제조를 위한 자재 소요량을 확인하려면, 자재 소요 계획(Material Requirement Planning)을 확인해야 합니다. 이 계획은 판매 주문(Sales Orders)에 기반한 **BOM 수량**을 기준으로 계산됩니다.

#### 4.3.1     구매 주문서 조회

메뉴 → 구매 주문 → 구매 주문서를 선택하여 구매 주문서 목록을 엽니다.

구매 주문 모듈은 그리드 테이블 형식으로 열립니다.

<figure><img src="../../../.gitbook/assets/image (52).png" alt=""><figcaption></figcaption></figure>

필터 버튼을 선택하거나 각 필드의 필터 박스에 조건을 입력합니다.

* 시작일 / 종료일: 조회 기간
* PO 상태
* 거래처 코드: F3 키를 눌러 드롭다운 목록에서 선택하거나 코드를 입력합니다.
* 품목 분류 코드: 품목의 분류, F3 키를 눌러 드롭다운 목록에서 선택
* 품목 코드: F3 키를 눌러 드롭다운 목록에서 선택하거나 코드를 입력합니다.

<figure><img src="../../../.gitbook/assets/image (53).png" alt=""><figcaption></figcaption></figure>

#### 4.3.2   구매 주문 생성

**새 구매 주문서**를 생성하는 경우, **신규** 버튼을 클릭합니다.

종합정보를 입력합니다.

\-        **참조 번호**: 자동으로 생성되며 변경할 수 없습니다

\-        **주문 생성일**: 주문 생성일.

\-        **공급업체 코드**: F3 눌러 공급업체 목록을 드롭다운으로 가져오거나 코드를 입력합니다.

\-       **통화 종류**: 이 거래에 적용되는 통화 종류(기본값은 VND). 다른 통화를 선택하면 환율을 입력해야 합니다.

\-        **납품 날짜**: 예상 납품 날짜.

\-        **납품 주소**: 납품 주소

\-        **결제 방법**: F3 눌러 드롭다운 목록에서 선택합니다.

\-        **담당자**: 담당자

\-       **설명 VN/EN/KR**: 언어로 된 설명

\-        **전화번호**: 전화번호

\-        **이메일**

또한 **추가 입력**을 클릭하여 추가 정보를 입력할 수 있습니다.

<figure><img src="../../../.gitbook/assets/image (54).png" alt=""><figcaption></figcaption></figure>

**세부 정보**\
**구매 주문(PO) 항목 입력**\
항목을 입력하는 방법은 두 가지가 있습니다:\
• **\[신규]** 버튼을 클릭하여 항목을 하나씩 개별적으로 추가하거나, **\[가져오기] 버튼**을 클릭하여 엑셀 파일에서 항목 목록을 업로드할 수 있습니다.\
• 항목을 개별적으로 추가하려면 \[신규] 버튼을 클릭하면, 세부 정보를 입력할 수 있는 팝업 창이 열립니다.

* **품목 코드**: F3 키를 눌러 드롭다운 목록에서 선택하거나 코드를 직접 입력
* **UOM (단위)**: 품목 목록에서 자동으로 불러옴
* **PO 수량**: 주문 수량
* **단가**: 해당 품목이 활성화된 단가표에 포함되어 있다면 자동으로 불러옴
* **세금 코드**: 품목 목록에서 자동으로 불러오며, 원하는 VAT 코드로 수정 가능
* **할인율 (%)**: 해당 품목에 대한 할인 비율
* **ETA**: 예상 도착일
* **ETD**: 예상 출발일
* **금액**: 자동 계산
* **세액**: 자동 계산
* **할인 금액**: 자동 계산

<figure><img src="../../../.gitbook/assets/image (55).png" alt=""><figcaption></figcaption></figure>

* 총금액: 자동 계산됩니다.

이미 엑셀 형식의 품목 목록 파일이 있는 경우에는 다음과 같이 진행합니다:

* \[템플릿] 버튼 선택: 올바른 형식의 파일을 받기 위해 선택합니다. 안내에 따라 각 열에 데이터를 입력합니다.
* \[가져오기] 버튼 선택: 파일을 업로드합니다. 파일 위치로 이동 → 파일 선택 → 열기 클릭
* 가져올 파일 이름을 선택합니다.

#### 4.3.3    구매 주문 가져오기

구매 주문 목록을 가져오려면 다음과 같이 진행합니다:

* **\[템플릿] 선택**: 올바른 형식의 파일을 받기 위해 선택합니다. 안내에 따라 Master 시트와 Detail 시트에 데이터를 입력합니다.
* **\[가져오기] 선택**: 데이터 가져오기 창을 엽니다.
* **\[Choose] 선택**: 파일을 업로드합니다. 파일 위치로 이동 → 파일 선택 → 열기 클릭
* **\[업로드] 선택**
* 오류 메시지(있는 경우)를 확인하고 수정합니다.

#### 4.3.4   구매 주문 수정

구매 주문 세부 사항은 상태가 '오픈(Open)'일 때 관련 아이콘 (바로 가기 및 아이콘 섹션) 을 사용하여 수정/삭제할 수 있습니다. '처리 중(Processing)' 또는 '완료(Completed)' 상태에서는 데이터가 잠깁니다. '취소(Cancel)'는 데이터가 제거됨을 의미합니다.

* 구매 주문 수정:\
  수정 아이콘 또는 구매 주문 번호를 클릭하여 세부 사항을 엽니다.\
  일반 정보(General info)를 직접 수정하거나, 각 항목 행의 수정 아이콘을 선택하여 세부 정보(Detail info)를 조정합니다.\
  저장(Save).
* 구매 주문 또는 항목 삭제:\
  삭제 아이콘을 선택하고 지침을 따릅니다.\
  여러 항목 또는 구매 주문을 제거해야 하는 경우, 각 행 앞의 체크박스를 선택 → 삭제(Delete) → 저장(Save).
