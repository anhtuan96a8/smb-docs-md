# 3.2  	가격표

판매 가격표는 기업이 고객에게 제공하는 각 제품의 가격을 의미합니다. 이 가격은 고객별, 제품 코드별 또는 프로모션 기간별로 다르게 설정될 수 있습니다.

<figure><img src="../../.gitbook/assets/image (89).png" alt=""><figcaption></figcaption></figure>

#### 3.2.1    새 가격표 생성 <a href="#toc191471943" id="toc191471943"></a>

메뉴 -> 판매 -> 가격표로 이동하여 가격표 목록 화면을 엽니다.

이 화면에서 사용자는 **생성일을** 변경하여 이전 가격표를 볼 수 있습니다(기본적으로 현재 날짜까지의 가격표가 표시됨). **각 필드별 조건을 필터링**하여 원하는 가격표를 검색할 수도 있습니다.

<figure><img src="../../.gitbook/assets/image (90).png" alt=""><figcaption></figcaption></figure>

새로운 가격표를 생성하는 경우 “신규”를 선택하여 가격표 입력 양식을 엽니다.

<figure><img src="../../.gitbook/assets/image (91).png" alt=""><figcaption></figcaption></figure>

_**종합정보:**_

·       가격표: 자동으로 생성되며 변경할 수 없음.

&#x20;       이전 가격표를 검토하려면 F3 눌러 목록에서 선택합니다

·       생성일: 가격표가 생성된 날짜.

·       고객 코드: F3 눌러 고객 목록을 표시하고 원하는 고객 코드를 선택.

·       발효일: 가격표가 적용되는 시작일

·       종료일: 가격표가 무효화되거나 중단되는 날짜.

·       통화: 가격표에서 거래된 통화(기본 가격: VND).

·       설명(VN/EN/KR): 베트남어/영어/한국어로 번역

&#x20;       참고: (\*)로 표시된 셀은 입력해야 합니다.

_**세부 정보**_

“신규”를 **눌러** 품목 입력 양식을 열고 입력 후 “저장”을 클릭.

·       품목 코드: F3 눌러 목록에서 선택 또는 코드 직접 입력.

·       계산 단위: 품목 코드에 따라 자동으로 표시되며 필요 시 F3 눌러 계산 단위 재선택.

·       단가: 품목 1단위당 가격

·       최소 주문 수량: 해당 가격이 적용되는 최소 주문 수량

·       최대 주문 수량: 해당 가격이 적용되는 최대 주문 수량

&#x20;       (예: 품목 A의 가격은 0-100 개 주문 시 VND10,000적용)

·       세율: 품목 코드에 따라 자동으로 표시되며 필요 시 F3 눌러 세율 재선택.

·       비고

_**엑셀 파일로 시스템에 자재 목록 업로드**_

사용자는 가격표에 여러 자재 코드를 한 번에 입력해야 하는 경우 ERP 시스템에 엑셀 파일로 자재 목록을 업로드할 수 있습니다.

\-        **샘플 파일 선택**: 엑셀 파일을 다운로드하여 지침에 따라 데이터를 해당 컬럼에 복사합니다.

\-        **입력 선택**: 엑셀 파일을 업로드합니다. 파일이 있는 폴더로 이동 -> 파일 선택 -> “Open” 클릭합니다.

&#x20;         일반적으로 엑셀 파일은 자동으로 업로드되며, 그렇지 않은 경우 파일 이름을 클릭합니다.

\-        업로드에 완료된 후에 오류 줄이 Error 항목에 표시되며 사용자는 정보를 수정하고 파일을 다시 업로드해야 합니다.

#### 3.2.2    동시에 여러 가격표 입력 방법 <a href="#toc201592041" id="toc201592041"></a>

사용자는 여러 가격표 목록을 가지고 있는 경우, 엑셀 파일 import 방법을 선택할 수 있습니다.

\-        **샘플 파일 선택**: 엑셀 샘플 파일을 다운로드합니다. 엑셀 파일(Master 및 Detail 시트 모두)의 지침에 따라 데이터를 해당 컬럼에복사합니다.

\-       **입력 선택**: 엑셀 파일을 업로드합니다. 파일이 있는 폴더로 이동 -> 파일 선택 -> “Open” 클릭.

일반적으로 엑셀 파일은 자동으로 업로드되며, 그렇지 않은 경우 파일 이름을 클릭합니다.

\-        업로드에 완료된 후 Erro Info 컬럼(Master 및 Detail)에서 검토하고 정보를 수정합니다. 다음에는 파일을 다시 업로드합니다.

#### 3.2.3    가격표 상세 수정 <a href="#toc201592042" id="toc201592042"></a>

가격표가 주문에 사용되지 않은 경우 가격표를 수정/삭제할 수 있습니다. 그러나 가격표가 주문을 생성에 사용된 경우, 수정/삭제가 불가능합니다

<figure><img src="../../.gitbook/assets/image (94).png" alt=""><figcaption></figcaption></figure>

가격표 세부 정보를 변경하는 경우 편집 아이콘 또는 가격표 번호를 선택하여 가격표 세부 정보를 엽니다. 종합정보 및/또는 세부 정보를 조정한 후 저장합니다.

#### 3.2.4    자재 및 고객별로 가격 필터링 <a href="#toc201592043" id="toc201592043"></a>

사용자가 (모든 고객의) 품목별로 가격을 검토할 경우, **메뉴 -> 판매 -> 가격표 세부 정보를 선택합니다.**

<figure><img src="../../.gitbook/assets/image (95).png" alt=""><figcaption></figcaption></figure>

각 필드의 필터 상자에 필터 조건을 입력하여 원하는 가격 정보를 확인합니다
