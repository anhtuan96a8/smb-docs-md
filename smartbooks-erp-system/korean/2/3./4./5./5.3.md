# 5.3 	완제품 입고

이 기능은 생산이 완료된 후 완제품을 입고 처리할 때 사용됩니다. 제조지시서(MO)와 연동되어 상태를 Open(대기)에서 **Processing(진행 중)** 또는 Completed(완료)로 업데이트합니다.

#### 5.3.1     완제품 입고 조회

메뉴 → 제조 → 제조지시서로 이동하여 MO 화면을 엽니다.

Filter 버튼을 선택하거나 각 필드의 필터 박스에 조건을 입력합니다.

* From date / To date: 조회 기간

#### 5.3.2  완제품 입고 생성

완제품 입고를 생성하려면, **\[New] 버튼을 선택**하여 입력 화면을 엽니다.

**일반 정보**

* Reference No.: 자동 생성됨
* Date: 날짜
* Note: 비고
* Description VN: 베트남어 설명
* PIC: 담당자
* Warehouse: 입고 창고, F3 키를 눌러 목록에서 선택
* Production Line: 생산 라인
* MO No.: F3 키를 눌러 목록에서 제조 지시서(MO) 선택, 선택 시 MO 수량이 불러와짐
* Machine Code: 기계 코드
* Molding Code: 금형 코드
* Shift Code: 근무조 코드
* Warehouse Issue Material: 자재 출고 창고
* Start Date/Complete Date/Duration: 시작일/완료일/소요 시간

**상세 정보**

* BOM에서 자동으로 불러옴 (품목 코드, 품목명, 단위, 창고 코드, 제조 수량, 제조 지시 번호)
* Business Quantity: 입고 수량. 수정 아이콘을 클릭하여 수량 입력
* MO Received Quantity: 자동 불러오기, 수령된 수량
* 노트

#### 5.3.3    완제품 입고 전표 수정

완제품 입고 전표는 **보류(Hold)** 상태일 때만 수정 및 삭제가 가능합니다.\
전표가 **확정(Released)** 상태가 되면 회계 모듈과 연동되므로 수정이 불가능합니다.

* **완제품 입고 전표를 수정하려면**, 수정 아이콘 또는 FG 입고번호를 선택하여 상세 화면을 엽니다.\
  일반 정보는 직접 수정할 수 있으며, 상세 정보는 각 항목 옆의 수정 아이콘을 선택하여 수정합니다.
* **입고 전표를 삭제하려면**, 삭제 아이콘을 선택하고 지시에 따라 진행합니다.
* **여러 개의 입고 전표를 삭제하려면**, 왼쪽 체크박스를 선택한 후 삭**제(Delete)** 를 클릭합니다.
