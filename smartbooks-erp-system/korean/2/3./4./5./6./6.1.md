# 6.1 품목 목록

품목 목록은 원자재부터 완제품까지의 품목 정보를 포함한 목록입니다.

**메뉴 -> 기본정보 -> 재고 관리로 이동하여 품목 목록을 엽니다.**

각 필드의 필터 박스에 조건을 입력하여 원하는 품목을 검색할 수 있습니다.

<figure><img src="../../../../../.gitbook/assets/image (23).png" alt=""><figcaption></figcaption></figure>

#### 6.1.1 품목생성

새로운 품목을 생성하려면 \[신규 품목]을 선택하여 입력 화면을 엽니다.

<figure><img src="../../../../../.gitbook/assets/image (24).png" alt=""><figcaption></figcaption></figure>

**일반 품목 정보**

* **품목 코드**: 필수 항목이며, 품목을 나타내는 문자 및 숫자를 포함해야 합니다.
* **설명 (베트남어/영어)**: 품목에 대한 설명 (색상, 사이즈, 브랜드 등)
* **분류 코드 (Class code)**: 품목의 분류, F3 키를 눌러 드롭다운 목록에서 선택
* **단위 (UOM)**: 측정 단위, F3 키를 눌러 드롭다운 목록에서 선택
* **그룹 단위 (Group UOM)**: 그룹에서 사용하는 단위
* **사용 여부 (Active)**: 해당 품목이 현재 사용 중인지 여부
* **상위 품목 여부 (Is Parent)**: 이 품목이 상위 품목이고 하위 품목 코드를 가지는 경우 선택
* **상위 품목 (Item Parent)**

**상세 정보**

* **일반 탭 (General tab)**

&#x20;      o  **구매 단위 (Purchase UOM)**: 구매 시 사용하는 단위

&#x20;      o  **판매 단위 (Sale UOM)**: 판매 시 사용하는 단위

&#x20;      o  **창고 코드 (Warehouse code)**

&#x20;      o  **다른 품목 코드 (Other Item code)**: 해당 품목이 다른 이름을 갖는 경우

&#x20;      o  **매입세 코드 / 매출세 코드 (Input/Output Tax code)**: 매입/매출 부가가치세 코드

* **계정 리스트 (Account List)**

&#x20;      o  **재고 계정 (Inventory Acct)**: 회계 시스템의 재고 계정

&#x20;      o **매출원가 계정 (COGS Acct)**: 회계 시스템의 비용 계정

&#x20;      o  **매출 계정 (Revenue Acct)**: 회계 시스템의 수익 계정

* **단위 변환 비율 (UOM Conversion Rate)**
* **최소 주문 수량 (MOQ)**

&#x20;      o **공급업체 코드 (Vendor code)**: F3 키를 눌러 드롭다운 목록에서 선택

&#x20;      o  **최소 주문 수량 (Minimum Order Quantity)**

&#x20;      o  **발주 리드 타임 (Order lead time)**: 구매 주문부터 입고까지 소요되는 시간

&#x20;      o  **기본 여부 (Default)**: 기본값 여부

* **기타 정보 (Other Info)**

#### 6.1.2  다수 품목 가져오기

품목 목록이 이미 있는 경우, 엑셀 파일을 통해 가져올 수 있습니다.

* **\[템플릿 선택] 버튼**: 올바른 형식의 템플릿을 다운로드합니다. 안내된 열(column)에 따라 데이터를 입력합니다.
* **\[가져오기] 버튼**: 파일을 업로드합니다. 파일 위치로 이동 → 파일 선택 → \[열기] 클릭
* 파일은 자동으로 업로드되거나, 파일 이름을 선택하여 업로드할 수 있습니다.

#### 6.1.3 품목 수정

\
기존 품목을 수정하려면 해당 품목의 상세 정보를 열어 일반 정보 및/또는 상세 정보를 변경할 수 있습니다. 이는 새로운 품목을 생성하는 방식과 유사합니다.
