# 6.2  재고 요청 목록

재고 요청(IR)은 **MO(작업 지시서) 화면**에서 직접 생성할 수 있으며(MO 항목 참조), 또는 메뉴에서 독립적으로 생성할 수도 있습니다.

#### 6.2.1  생성

**\[신규]** 버튼을 선택하여 입력 화면을 엽니다

<figure><img src="../../../../../.gitbook/assets/image (27).png" alt=""><figcaption></figcaption></figure>

**일반 정보**

* **IR 번호 (IR No.)**: 자동 생성
* **MO 번호 (MO No.)**: F3 키를 눌러 목록을 불러온 후, 하나 또는 여러 개의 MO를 선택
* **요청일 (Date)**: 재고 요청 날짜
* **출고 창고 / 입고 창고 (From warehouse / To warehouse)**: 재고를 출고하는 창고 / 입고받는 창고
* **공장 코드 (Plant code)**: F3 키를 눌러 목록에서 선택
* **완제품 입고 메모 (FG Receipt Note)**: 생산 창고에서 완제품 창고로 이동할 경우 작성
* **담당자 (PIC)**: 책임자
* **비고 (Note)**

**상세 정보**\
&#xNAN;**\[신규]** 버튼을 클릭하여 항목을 추가

* **품목 코드 (Item Code)**
* **요청 수량 (QTY Requested)**
* **단위 (UOM)**
* **비고 (Note)**

#### 6.2.2  IR 수정

IR 상태가 **열림(Open)** 상태일 때만 수정 또는 삭제할 수 있습니다. **처리 중(Processing)** 또는 **완료(Completed)** 상태일 경우 데이터는 잠금 상태가 됩니다.

IR을 수정하려면 **수정 아이콘 또는 IR 번호**를 선택하여 상세 화면을 엽니다.

* 일반 정보는 직접 수정할 수 있으며, 상세 정보는 **수정 아이콘**을 선택하여 새 요청 생성 방식과 유사하게 변경
* 품목 상세 정보는 **수정 아이콘 또는 삭제 아이콘**을 선택하여 수정/삭제 가능
* 여러 개의 IR을 삭제하려면 각 행의 좌측 체크박스를 선택한 후 **\[삭제(Delete)]** 클릭
