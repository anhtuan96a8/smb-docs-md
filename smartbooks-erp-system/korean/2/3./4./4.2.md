# 4.2	공급업체 가격표

공급업체 가격표는 공급업체가 제공한 자재, 단가, 결제 조건, 할인, 배송 정보 등과 관련된 정보를 포함합니다. 가격표 생성은 자주 주문을 생성할 때 각 품목의 가격 정보를 찾아보는 시간을 절약할 수 있는 데 도움이 됩니다.

<figure><img src="../../../.gitbook/assets/image (46).png" alt=""><figcaption></figcaption></figure>

#### 4.2.1     가격표 찾기

메뉴 -> 구매 -> **공급업체 가격표**를 선택하여 가격표 목록을 엽니다.

목록은 그리드 형식으로 표시되며, 사용자가 각 열에서 조건을 쉽게 필터링할 수 있습니다.

이 외에도 이 모듈에는 사용자가 각 가격표를 열지 않고도 세부 정보를 필터링할 수 있는 필터 버튼이 있습니다

<figure><img src="../../../.gitbook/assets/image (47).png" alt=""><figcaption></figcaption></figure>

_**정보별 필터링**_

·       사용 중: 가격표의 상태(사용 또는 미사용)

·       가격표: 가격표 번호(기억이 나는 경우)

·       공급업체 코드: F3 눌러 목록을 가져오거나 코드를 입력합니다.

·       코드: F3 눌러 목록을 가져오거나 코드를 입력합니다.

·       유효 날짜: 가격표 적용 날짜

·       유효 종료 날짜: 가격표가 더 이상 유효하지 않은 날짜

#### 4.2.2     가격표 생성

가격표를 생성하는 경우, **신규**를 선택하여 입력 창을 엽니다

<figure><img src="../../../.gitbook/assets/image (48).png" alt=""><figcaption></figcaption></figure>

_**종합정보**_

·     **가격표**: 가격표 번호 추가(문자와 숫자 포함, 사이에 공백 없이)

·   **설명 VN**: 베트남어로 설명

·       **사용 중**: 이 가격표가 사용 중인 경우 사각형 상자를 클릭합니다.

·       **비고**

주의사항:(\*)가 있는 필드는 필수 입력 항목입니다.

_**세부 정보**_

**신규**를 클릭하여 자재 입력 양식을 엽니다.

\-        자재 코드: F3 눌러 목록을 가져오거나 코드를 입력합니다.

\-        공급업체 코드: F3 눌러 목록을 가져오거나 코드를 입력합니다.

\-        창고 코드: 자재 수령 창고. 자재 목록에서 자동으로 불러옵니다. 변경하는 경우 F3 눌러 목록에서 선택          하거나 코드를 입력합니다.

\-        통화 단위: 자동으로 표시

\-        최소 수량: 가격 범위 내 최소 주문 수량

\-        최대 수량: 가격 범위 내 최대 주문 수량

\-        통화: 거래 통화: VND가 기본값으로 설정됩니다.

\-        단가: 자재 1단위의 가격

\-        유효 날짜: 가격표 적용 날짜

\-        종료 날짜: 가격표가 더 이상 유효하지 않은 날짜

\-        비고

<figure><img src="../../../.gitbook/assets/image (49).png" alt=""><figcaption></figcaption></figure>

_**만약 아이템 목록이 있는 경우**:_

\-         **템플릿 가져오기**: 엑셀 양식을 받기 위해 '템플릿' 버튼을 누르고, 안내에 따라 데이터를 입력합니다.

\-         **가져오기**: '가져오기' 버튼을 눌러 파일을 업로드합니다. 파일 위치로 이동하여 파일을 선택한 후 '열기'를 클릭합니다.

\-         **가져오기 실행**

<figure><img src="../../../.gitbook/assets/image (50).png" alt=""><figcaption></figcaption></figure>

#### 4.2.3     여러 가격표 동시에 입력

사용자가 공급업체 단가 목록이 있는 경우, 엑셀 파일을 가져오는 방식으로 더 빠르게 등록합니다.

\-        탬플릿 버튼을 크릭하여 엑셀 양식을 다운로드하고, 안내에 따라 각 열에 필요한 정보를 입력합니다.

\-        .가져오기 버튼을 클릭하여 파일을 업로드합니다.

<figure><img src="../../../.gitbook/assets/image (51).png" alt=""><figcaption></figcaption></figure>

\-         **Choose’ 선택** → 파일 위치로 이동 → 파일 선택 → ‘열기’ 클릭

\-         **‘업로드’ 선택**

\-       **Master** 및 **Detail** 탭에서 오류 정보(있는 경우)를 확인하고, 올바른 정보로 수정

\-         **데이터 업로드 실행**

#### 4.2.4     공급업체 가격표 상세하게 수정.

단가표가 아직 어떤 주문에도 사용되지 않은 경우에는 \[바로가기 및 아이콘] 섹션의 관련 아이콘을 사용하여 세부 정보를 수정하거나 삭제할 수 있습니다.

**참고**: 단가표가 이미 사용된 경우에는 어떠한 정보도 수정할 수 없습니다.

#### 4.2.5     단가표 삭제/출력

단가표를 수정하려면 다음과 같이 진행합니다:

\-        공급업체 단가표 화면에서 수정하려는 단가표의 편집 아이콘 또는 단가표 번호를 클릭하여 선택합니다.

\-        .공급업체 단가표 입력 화면에서 일반 정보는 직접 수정하고, 세부 정보는 편집 도구(편집,복사,삭제)를 사용하여 변경합니다
