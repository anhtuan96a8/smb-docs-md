# 4.1	공급업체 목록

본 모듈은 S\&S ERP 시스템에서 구매 주문을 생성하는 기본 프로세스를 제공합니다. 구매 주문 프로세스는 다음 단계로 구성됩니다:

·       공급업체 가격표

·       구매 요청서

·       구매 주문서

·       PO 단가 업데이트

·       구매 입고

·       구매한 상품 반품

·       매입 세금계산서 업데이트

### 4.1        공급업체 목록

공급업체 목록은 공급업체와 관련된 정보를 저장하는 곳으로, 여기에는 기업명, 세금 코드, 거래 통화, 회계 계좌가 포함됩니다.

<figure><img src="../../../.gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure>

#### 4.1.1     공급업체 생성

메뉴 -> 마스터 데이터 -> **공급업체 목록을** 선택하여 정보 목록을 엽니다.

목록은 그리드 형태로 표시되어 사용자가 열에서 조건을 쉽게 필터링할 수 있습니다

<figure><img src="../../../.gitbook/assets/image (42).png" alt=""><figcaption></figcaption></figure>

새 공급업체 생성을 위해 **신규**를 선택하여 입력 양식을 엽니다.

_**종합정보**_

·       공급업체 코드: 각 공급업체의 유일한 코드입니다.

·       공급업체명(VN/EN/KR): 다양한 언어로 된 공급업체명(기업명 또는 개인 이름)

·       라벨: LOT 번호(있는 경우)

<figure><img src="../../../.gitbook/assets/image (43).png" alt=""><figcaption></figcaption></figure>

·       구매 주문서: 해당 공급업체의 모든 주문 종합(주문 수량, 금액, 세금, 할인 등)

·       입고 전표: 해당 공급업체로부터의 입고 전표 목록.

·       활동: 고객이 아직 활동 중이며 협력하고 있는지 여부.

_**세부 정보**_

·       **탭 1. 공급업체**

&#x20;        o   대표자: 법인장 또는 이사 또는 담당자

&#x20;        o   주소: 회사 주소

&#x20;        o   전화번호: 회사 전화번호(있는 경우)

&#x20;        o   팩스 번호

&#x20;        o   세금코드: 세금코드 또는 회사코드(있는 경우)

·       **기본 탭**

&#x20;        o   거래 통화: 기본적으로 VND

&#x20;        o   지불 기한: 지불 조건(자세한 내용을 보려면 F3을 누릅니다)

&#x20;        o   세율: 부가세 코드

&#x20;        o   미지급금 계좌(AP): 외상매입금 계좌(있는 경우)

&#x20;        o   선지급 계좌: 선지급 계좌(있는 경우)

·       **기타 정보 탭**

&#x20;        o   공급업체의 은행계좌 정보

#### 4.1.2      여러 공급업체 동시에 입력

사용자가 여러 공급업체 목록을 가지고 있는 경우, 데이터를 입력하는 시간을 절약하기 위해 임포트 방법을 선택할 수 있습니다.

\-        **템플릿 파일 선택**: 엑셀 템플릿을 가져옵니다. 지침에 따라 열에 데이터를 입력합니다.

\-        **입력 선택**: 엑셀 파일을 업로드합니다.

&#x20;        o   파일 저장 위치로 들어가서 -> 파일 선택 -> 열기

&#x20;             일반적으로 파일은 자동으로 표시되지만, 표시되지 않는 경우 파일 이름을 클릭합니다.

\-        오류 정보를 확인하고 데이터를 적절하게 수정합니다. 필요하면 파일을 다시 업로드합니다.

\-        **불러오기**.

<figure><img src="../../../.gitbook/assets/image (44).png" alt=""><figcaption></figcaption></figure>

#### 4.1.3     공급업체 수정

공급업체가 기업 정보를 변경하는 경우, 사용자는 ERP 시스템에서 직접 정보를 수정할 수 있습니다.

공급업체 목록에 들어가서 수정 아이콘이나 공급업체 코드를 클릭하기를 통해 수정할 공급업체를 선택합니다. 종합정보 및/또는 세부 정보 항목에서 적합한 정보를 수정합니다. 이는 새로운 공급업체를 생성할 때와 유사합니다

<figure><img src="../../../.gitbook/assets/image (45).png" alt=""><figcaption></figcaption></figure>

#### 4.1.4     공급업체 목록 내보내기

공급업체 목록의 엑셀 파일이 필요한 경우, 내보내기(export)를 선택하시오. 파일은 로컬 위치에 저장됩니다.
