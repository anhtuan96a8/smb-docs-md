# 4.7	매입 세금계산서 업데이트

입고 전표가 생성되면, 결제 절차를 위해 세금계산서도 함께 수령하게 됩니다. 해당 세금계산서는 회계팀에서 업데이트합니다.

#### 4.7.1     세금계산서 매칭

검토 기간을 변경하여 지난 입고 전표를 확인합니다 (기본값은 해당 월).

세금계산서가 아직 업데이트되지 않은 항목의 체크박스를 선택한 후 \*\*매입세금계산서 매칭(Matching Input Invoice)\*\*을 진행합니다.

<figure><img src="../../../.gitbook/assets/image (6).png" alt=""><figcaption></figcaption></figure>

기본 정보

* 공급업체 코드(Vendor code): 자동 불러오기
* 공급업체명 (베트남어)(Vendor name - VN): 자동 불러오기
* 공급업체명 (영어)(Customer name - EN): 자동 불러오기
* 일련번호(Serial No.): 세금계산서 일련번호
* 세금계산서 발행일자(Invoice date)
* 세금계산서 번호(Invoice No.)
