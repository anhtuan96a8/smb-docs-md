# 3.1 Customer list

This Sales Order (SO) module provides the basic procedure to create a client’s order. The SO process includes these steps:

·     Create **Sales price**: inform it to the customers

·     Create **Sales order**:  if the customers agree with the prices, they’ll make the orders

·     Update **Sales price**

·     Create **Delivery note**:  when the goods is ready to release.

·     Create **Sales return** (if any)

·     **Matching Output Invoices**

### 3.1.  Customer List

Customer list is summary information about all customers that you collect to make the sales documents

#### 3.1.1                   Create a new customer

To create a new customer, go to **Menu -> Master Data -> Customer List**

<figure><img src="../../.gitbook/assets/image (71).png" alt=""><figcaption></figcaption></figure>

The Customer List window opens as the grid table and filter boxes. If you need to find a customer on the list, give the conditions in the filter boxes at the top of the column

Select **New** to input new customer info

_**General information**_

·     **Customer code**: unique code for each client

·     **Customer name**: client name (business name or personal name)

·     **Currency code**: currency using in the transaction. If a client uses 2 currencies, you have to make 2 different Customer codes for each.

·     **Activity**: The customer is still active and continues to buy and sell with the company.

·   **Sales Order**:  Summary of all the sales orders (order quantity, amount, tax, discount…)

·     **Delivery**: List of delivery notes

#### 3.1.2                  Import many customers

If you have a list of many customers, then import the excel file is a good solution.

\-     Select **Template**: to get the excel template. Copy data by columns as the instructions

\-     Select **Import**: to upload the excel file.

o  Go to the **file location -> choose the file** **-> Open**

o  Usually, the file is read automatically, otherwise, click the file name.

o  Check the Error messages (if any) and adjust the wrong data. **Import/ Upload Data.**

#### 3.1.3                   Edit customer information

If the customers change their business info, it is very easy to adjust them on S\&S ERP.

Similar to add a new customer, Go to **Menu -> Master Data -> Customer List.**

\-     When the Customer List shows up, choose the particular client by clicking their code or edit icon.

\-     On the detail page, adjust the data as you want (except the Customer code).
