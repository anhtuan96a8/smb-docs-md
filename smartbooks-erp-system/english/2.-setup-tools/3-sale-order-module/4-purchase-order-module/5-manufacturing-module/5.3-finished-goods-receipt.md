# 5.3 Finished Goods Receipt

This is used to make the receipt of finish goods after the production. It links to MO to update its

status from Open to Processing/ Completed.

#### 5.3.1                Search for a FG Receipt

Go to **Menu -> Manufacturing- -> Manufacturing Order** to open the MO window.

Select Filter button or give the conditions to the filter box of each field

From date/ To date: review period

#### 5.1.1.                 Edit FG Receipt

FG receipt can be edited/ delete when it is on the Hold status. If it is released, its data is locked since it interfaces with the accounting part.

To edit the FG Receipt, select the edit icon or FG receipt number to open its detail. Change the general info directly and/or detail info by selecting the edit icon.

To delete the FG receipt, select the delete icon and follow the instructions.

To delete many FG receipts, check the boxes on the left -> **Delete**
