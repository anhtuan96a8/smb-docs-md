# 6.3 Stock Transfer

Stock transfer is used when you need to move the item from this warehouse to another warehouse. For example, from the raw materials to the factories or from the manufacturing line to the finished good warehouse.

#### 6.3.1                 Create Stock Transfer

Go to Menu -> Inventory Management -> Stock Transfer to open the transfer list.

<figure><img src="../../../../../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>

You can change the review period (From date/ To date) to view older transfers.

To create a new transfer, select **New** to open input form

<figure><img src="../../../../../.gitbook/assets/image (3).png" alt=""><figcaption></figcaption></figure>

Enter the required information (fields with \* are mandatory). If the information in the inventory

transfer request is entered, the items will be automatically loaded from the receipt. The user should

check and adjust the quantities to match the actual stock.

&#x20;

To add item details, click **New** and enter the required information. If the item has a LOT number, you

need to update the LOT number before saving the receipt

#### 6.3.2                 Edit Stock Transfer

If the transfer is on the hold status, it is able to edit its info. When it is posted, the data is locked.

To edit the stock transfer, select the edit icon or the transfer number to open its detail. Change the

general info and/or the detail info, similar to create a new transfer -> **Save**.

&#x20;

**Delete/ print stock transfer**

\-     To delete a stock transfer or an item, select the delete icon.

\-     To delete many transfer or many items, check the left boxes of each row à Delete

\-     To print the transfer, select the print icon on each row
