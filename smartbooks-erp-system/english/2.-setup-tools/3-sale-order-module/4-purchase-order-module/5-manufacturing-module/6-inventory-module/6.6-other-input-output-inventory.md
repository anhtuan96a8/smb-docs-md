# 6.6 Other input/ Output Inventory

Other Input/ Output Inventory is used to increase or decrease the stocks with other reasons, such as

materials returns or release raw materials to outsourcing services.

#### 6.6.1.                 Search for Other Input/ Output Inventory

Go to Menu -> Inventory Management -> Other Input/ Output Inventory to open the list of vouchers.

Select Filter button or give the conditions to the filter box on top of each field.

.         From date/ To date: review period

#### 6.6.3                Import Other Input/ Output Inventory

If you have an excel file of items, you can import it to save time

\-     Select **Template**: to get the correct format. Add the info by columns as the instructions.

\-     Select **Import**: to open import window.

\-     Select **Choose**, go to the file location -> choose the file -> Open -> **Upload**

\-     Check the error messages (if any) and correct them

\-     **Upload file/ Import**

#### 6.6.4                 Edit Other Input/ Output Inventory

If the Other Input/ Output voucher is on the hold status, it is able to edit its info. When it is posted, the data is locked.

To edit an Other Input/ Output voucher, select the edit icon or the voucher number to open its detail. Change the general info and/or the detail info, similar to create a new Other Input/ Output voucher -> **Save.**

<figure><img src="../../../../../.gitbook/assets/image (100).png" alt=""><figcaption></figcaption></figure>

**Delete/ print Stock Adjustment**

\-     To delete a voucher or an item, select the delete icon.

\-     To delete many vouchers or many items, check the left boxes of each row -> **Delete**

\-     To print the voucher, select the print icon on each row

\-     To export excel file, check the left boxes on each row -> **Export**
