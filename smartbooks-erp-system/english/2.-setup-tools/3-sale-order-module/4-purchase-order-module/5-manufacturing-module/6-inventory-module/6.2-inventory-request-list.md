# 6.2 Inventory Request List

The Inventory Request (IR) can be made directly from the MO window (see MO for reference) or you

can make it independently from the menu.

<figure><img src="../../../../../.gitbook/assets/image (9).png" alt=""><figcaption></figcaption></figure>

#### 6.2.1                 Create IRs

Select the **New** to open the input form

_**General info**_

\-     IR No.: auto created

\-     MO No.: press F3 to get the dropdown of list, select the one (or more) MO(s)

\-     Date: requested date

\-     From warehouse/ To warehouse: the warehouse releases/ receives the stocks

\-     Plant code: Press F3 to select from the list.

\-     FG Receipt Note: In case of transferring from the production warehouse to the finished goods warehouse.

\-     PIC: person in charge

\-     Note

_**Detail Info**_

Click **New**

\-     Item Code

\-     QTY Requested

\-     UOM

\-     Note

#### 6.2.2                Edit IRs

You can edit/delete an IR when it is on the Open status. If it is Processing or Completed, data is locked.

To edit the IR request, select the edit icon or IR number to open its detail.

\-     Change the general info directly and/or detail info by select the edit icon, similar to create a new request.

\-     Edit/ Delete the item detail by selecting the edit icon (delete icon)

\-     To delete many IR, check the box on the left of each row -> Delete
