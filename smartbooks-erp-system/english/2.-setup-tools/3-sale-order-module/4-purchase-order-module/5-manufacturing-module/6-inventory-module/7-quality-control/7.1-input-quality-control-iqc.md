# 7.1 Input Quality Control (IQC)

Quality control plays a very important role in the process of material flow in production and trade. To ensure the quality of purchased materials, it must go through IQC (Input Quality Control) and ensure the quality of goods before being sold through OQC (Output Quality Control).

### 7.1  Input Quality Control (IQC) <a href="#toc201246923" id="toc201246923"></a>

<figure><img src="../../../../../../.gitbook/assets/image (101).png" alt=""><figcaption></figcaption></figure>

The list of receipt notes from production is displayed on the screen below in a table format, allowing

users to filter by the desired fields.

Select the receipt note to check QC → Enter QC data

<figure><img src="../../../../../../.gitbook/assets/image (102).png" alt=""><figcaption></figcaption></figure>

We need to input this information in layout

_**General**_

\-     QC Pass WH

\-     Not-Good WH

\-     Inspector

\-     Inspection Date

\-     Evaluation

_**Detail**_

Click the **Edit** button for each LOT number to enter the QC evaluation details

<figure><img src="../../../../../../.gitbook/assets/image (103).png" alt=""><figcaption></figcaption></figure>

Select the inspection quantity, inspection date, and inspector.\
If the product passes QC, click the **QC Passed** button.\
If the product fails, enter the defect details and the quantity of defective items in the fields below.\
If you need to upload images of defective items, click the image icon to load the images into the system.
