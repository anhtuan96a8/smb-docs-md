# 6.4 Stocktaking

Stocktaking is used when you need check the actual quantity and compare to its booking quantity.

In some cases, there are the differences and you need to do the adjustments

#### 6.4.1                Create Stocktaking

Go to Menu -> Inventory Management -> Stocktaking to open the list.

<figure><img src="../../../../../.gitbook/assets/image (4).png" alt=""><figcaption></figcaption></figure>

You can change the review period (From date/ To date) to view older checking.

To create a new stocktaking, select **New** to open the input form

<figure><img src="../../../../../.gitbook/assets/image (5).png" alt=""><figcaption></figcaption></figure>

_**General info**_

\-     BatNbr: auto created

\-     Check Date

\-     PIC: person in charge

\-     Description VN/EN/KR

_**Detail info**_

Select the **New** to add the items

\-     Item code: press F3 to get the dropdown list or type the code

\-     Warehouse code: press F3 to get the dropdown list or type the code

\-     UOM: auto loaded

\-     Stock Quantity: auto loaded

\-     Actual Quantity: actual counted quantity

\-     Variance quantity: auto calculated

\-     Note

**Save**

#### 6.4.2                 Edit Stocktaking

If the Stocktaking is on the hold status, it is able to edit its info. When it is posted, the data is locked.

To edit the stocktaking, select the edit icon or the voucher number to open its detail. Change the

general info and/or the detail info, similar to create a new stocktaking -> **Save.**

<figure><img src="../../../../../.gitbook/assets/image (6).png" alt=""><figcaption></figcaption></figure>

**Delete/ print stock transfer**

\-     To delete a stocktaking or an item, select the delete icon.

\-     To delete many vouchers or many items, check the left boxes of each row -> **Delete**

\-     To print the voucher, select the print icon on each row

\-     To export excel file, check the left boxes on each row -> **Export**
