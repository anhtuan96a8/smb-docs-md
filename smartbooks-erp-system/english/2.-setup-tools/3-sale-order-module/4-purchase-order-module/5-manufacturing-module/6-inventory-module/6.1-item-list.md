# 6.1 Item list

### 6.1.  Item List

Item List is a list of item information including the materials to finished goods.

&#x20;

Go to **Menu -> Master Data -> Inventory Management** to open the Item List.

You can give the filter condition in the filter box of each field to search for any items

#### 6.1.1.                 Create an Item

To create a new item, select **New Item** to open the input form

<figure><img src="../../../../../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

_**General Item**_

\-     Item code: must have, includes characters and numbers representing the item

\-     Description VN/EN: describe about the item (color, size, branch…)

\-     Class code: class of items, press F3 to get the dropdown list

\-     UOM: unit of measure, press F3 to get the dropdown list

\-     Group UOM: UOM in group

\-     Active: item is still used

\-     Is Parent: Select if this a parent item and if it has order child item codes

\-     Item Parent

_**Detail Info**_

\-     General tab

o  Purchase UOM: UOM used for purchase items

o  Sale UOM: UOM used for sales items

o  Warehouse code

o  Other Item code: in case that item has 2 names

o  Input Tax code/ Output tax code: input/output VAT

&#x20;

\-     Account List

o  Inventory Acct: inventory account in accounting system

o  COGS Acct: expense account in accounting system

o  Revenue Acct: revenue account in accounting system

&#x20;

\-     UOM Conversion Rate

\-     MOQ

o  Vendor code: press F3 for getting the dropdown list

o  Minimum Order Quantity

o  Order lead time: time needed from purchase order to receive the items

o  Default: is default or not

\-     Other Info

&#x20;

After input the necessary info -> Save.

#### 6.1.2.                 Import Many Items

If you have a list of item, you can import the excel file.

\-     Select Template button: to get the correct format. Add data by columns as the instructions

\-     Select Import button: to upload the file. Go to the file location à choose the file à Open

\-     The file is auto uploaded or select the file name.

<figure><img src="../../../../../.gitbook/assets/image (8).png" alt=""><figcaption></figcaption></figure>

#### 6.1.3                 Edit an Item

You can edit the existed item by open the item detail à change the general info and/or detail info, similar to create a new item
