# 5.1 BOM List

This module provides the manufacturing flow from releasing raw material to receiving finished good

### 5.1.  BOM List

In this part, you can create the BOM for the manufacturing. It links to the Manufacturing Order, and

used to compare the actual and planned material quantity

<figure><img src="../../../../.gitbook/assets/image (45).png" alt=""><figcaption></figcaption></figure>

#### 5.1.1.                 Search for a BOM

Go to Menu -> Manufacturing -> BOM List to open the BOM window.

Select Filter button or give the conditions to the filter box of each field

\-  **Activated**: BOM status (active or inactive)

\-   **Item code**: press F3 to get the dropdown list or type the code

\-    **Class code**: press F3 to get the dropdown list or type the code

<figure><img src="../../../../.gitbook/assets/image (46).png" alt=""><figcaption></figcaption></figure>

#### 5.1.1.                 Import BOM(s)

To import the list of BOM

\-     Select **Template**: to get the correct format file. Add data into Master, Detail Component and Detail Operation sheet as the instructions.

\-     Select **Import**: to open Import Data window

\-     Select **Choose**: to upload the file, go to the file location à choose the file à Open

\-     Select **Upload**

\-     Check the error messages (if any) and correct them.

\-     Upload Data

<figure><img src="../../../../.gitbook/assets/image (50).png" alt=""><figcaption></figcaption></figure>

#### 5.1.4                 Edit BOM

You can edit/delete the BOM detail when it is not used to make any Manufacturing Order.

\-     To edit BOM, click the edit icon or BOM number to open the its detail.

&#x20;      o  Adjust the General info directly and/or Detail info by selecting the edit icon in each item row. **Save**

&#x20;      o  To delete the item, click the delete icon on that row and follow the instructions.

\-     To delete BOM, click the delete icon on that row and follow the instructions
