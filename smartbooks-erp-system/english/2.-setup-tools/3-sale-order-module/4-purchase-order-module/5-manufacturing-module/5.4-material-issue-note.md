# 5.4 Material Issue Note

This is used to make the issue note of material for the production. It links to MO to update its status from Open to Processing/ Completed.

Go to **Menu -> Manufacturing -> Material Issue** Note to open the its list.

Select Filter button or give the conditions to the filter box of each field

.        From date/ To date: review period

#### 5.4.1                 Create a Material Issue Note

To create a Material issue note, select **New** to open the input form

<figure><img src="../../../../.gitbook/assets/image (59).png" alt=""><figcaption></figcaption></figure>

_**General info**_

\-     RefNbr.: auto created

\-     Date

\-     Warehouse Code: Received warehouse, press F3 to get the dropdown list

\-     Production Line

\-     MO No.: press F3 to get the dropdown list, fill the MO to load the MO quantity.

\-     Note

_**Detail info**_

\-     Auto loaded from BOM (Item code, Item name, UOM, Warehouse code, FG code, Receipt Note, MO No.)

\-     **Business Quantity**: stock output quantity. Select edit icon to add the number.

\-     Note

\-     If you want to add more item, select **New** to open the input form.

**Save.**



#### 5.4.2                Edit Material Issue Note

Material Issue Note can be edited/ delete when it is on the Hold status. If it is released, its data is locked since it interfaces with the accounting part.

To edit the Issue Note, select the edit icon or Issue number to open its detail. Change the general info directly and/or detail info by selecting the edit icon.

&#x20;

To delete the Material Issue Note, select the delete icon and follow the instructions.

To delete many Material Issue Notes, check the boxes on the left -> **Delete**
