# 4.2 Vendor Price List

Vendor price list is the price information about item, prices, payment conditions, discount, delivery

info, etc. that the vendor provides you. The price list will save your time when you have to create the

purchase order regularly.

<figure><img src="../../../.gitbook/assets/image (23).png" alt=""><figcaption></figcaption></figure>

#### 4.2.1                Search a price list

Go to **Menu -> Purchase Order -> Vendor Price List** to open the list. it is shown up as a grid table.

Select the Filter button to open the requirements or type directly the condition in the filter box of each fields

<figure><img src="../../../.gitbook/assets/image (24).png" alt=""><figcaption></figcaption></figure>

_**Filter info**_

·     **Activated**: the status of the price list (active or inactive)

·    **Price List No.**: price list number (if remember)

·     **Vendor Code**: press F3 to get the dropdown list or type the code

·  **Item code**: it will show all of vendors that sell that item

·     **Validated date**: the date the price list is applied

·    **Expired date**: the date the price list is inactive.

#### 4.2.2                 Create a price list

To create a vendor price list, select **Add New** to open the input form

<figure><img src="../../../.gitbook/assets/image (25).png" alt=""><figcaption></figcaption></figure>

_**General Info:**_

·   **Price List No**.: add the price list number (including characters and number, no space in between)

·     **Description VN**: description in Vietnamese

·     **Activated**: check the box if it is active.

·     **Note**: comments or remarks

Note: (\*) fields are must.

_**Detail info**_

Select Add New to open the item info form.

\-    **Item code**: press F3 to get the dropdown list or type the code

\-     **Vendor code**: press F3 to get the dropdown list or type the code

\-  **Warehouse**: Warehouse to receive the stocks. Auto loaded from the item list. To change it, press F3 to get the dropdown list or type the code.

\-     **UOM**: auto loaded

\-     **Quantity from**: minimum order quantity for the price range

\-    **Quantity to**: maximum order quantity for the price range

\-     **Currency**: transaction currency. VND as default.

\-     **Unit price**: price for a unit

\-  **Validated date**: the date the price list is applied

\-     **Expired date**: the date the price list is inactive

\-     **Note**

<figure><img src="../../../.gitbook/assets/image (26).png" alt=""><figcaption></figcaption></figure>

If you have a list of items:

\-     Press **Template**: to get the excel form, add the data as the instruction.

\-     Press **Import** to upload file. Go to the file location -> choose the file -> Open.

\-    **Import.**

<figure><img src="../../../.gitbook/assets/image (27).png" alt=""><figcaption></figcaption></figure>

#### 4.2.3                 Import price list

If you have a list of vendor prices, then you can import the excel file as a faster way.

\-     Press **Template**: to get the excel format, input the necessary information by columns as the instructions

\-     Press **Import**: to upload the file.

<figure><img src="../../../.gitbook/assets/image (28).png" alt=""><figcaption></figcaption></figure>

\-     Select **Choose** -> go to the file location -> choose the file -> **Open.**

\-     Select **Upload**

\-     Check the **Error info** (if any) in Master and Detail tabs and adjust to the correct info.

\-     **Upload data.**

#### 4.2.4                 Edit the price list detail

When the price list has not used to make any orders yet, it is available to edit/delete its detail by using the related icons (Shortcut and Icons section)

**Note**: if the price list has already used, it does not allow to change any information.

#### 4.2.5 Delete/Print Price List

To make a change of the price list:

\-     In the Vendor Price List, choose the particular price list by clicking edit icon or price list number.

\-     In the Vendor Price List Input, change the general info directly and/or detail info by edit tool (Edit/ Copy/ Delete).
