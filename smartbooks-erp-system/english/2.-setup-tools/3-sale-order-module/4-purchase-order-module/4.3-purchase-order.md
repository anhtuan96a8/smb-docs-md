# 4.3 Purchase Order

Purchase Order is used to record the orders you ask from the vendors. It is linked to the Vendor Price List (if you’ve already input the prices of these items from that vendor), so that the prices are automatically loaded. You can also adjust the prices and add the discounts (if any).

To get the material requirement for manufacturing, you should check the Material Requirement Planning since it is calculated BOM quantity from the Sales Orders.

#### 4.3.1                 Search for a Purchase order

\
Go to **Menu -> Purchase Order -> Purchase Order** to open the list of purchase order.

The Purchase Order module is opened as the grid table.

Select the Filter button or type the conditions in the filter box of each field.

\-     **From date / To date**: the review period

\-     **PO status**

\-    **Vendor code**: press F3 to get the dropdown list or type the code.

\-    **Class code**: class of items, press F3 to get the dropdown list

\-    **Item code**: press F3 to get the dropdown list or type the code

#### 4.3.2                 Create a Purchase order

To create a new Purchase Order, select **New** to open the purchase order input.

_**General info**_

\-     **Reference No.**: automatically create, no changeable.

\-   **PO Date**: created date

\-     **Vendor code**: press F3 to get the dropdown list or type the code.

\-    **Currency code**: currency applied in this transaction (VND as default). If you choose other currency, you have to input the exchange rate.

\-     **Received date**: the date the goods are arrived

\-     **Received address**: the destination to get the goods

\-   **Payment term**: press F3 to get the dropdown list of terms

\-     **Contact name**: vendor contact person

\-   **Contact number**

\-   **Contact email**

\-    **Description VN/EN/KR**

<figure><img src="../../../.gitbook/assets/image (31).png" alt=""><figcaption></figcaption></figure>

_**Detail info**_

Enter Items for PO

There are 2 ways to enter items:

·     Click **New** button (to add each item individually) or click the **Import** button to upload a list of items from an Excel file.

·     To add each new item, click the **New** button, and the system will open a pop-up where you can enter the details.

\-     **Item code**: press F3 to get the dropdown list or type the code

\-     **UOM**: loaded automatically from Item list

\-     **PO Quantity**:  order quantity

\-     **Unit Price**: auto loaded if the item is in the active price list.

\-     **Tax code**: auto loaded from the item list, can edit the desired VAT code.

\-    **% Discount**: discount percent for that item.

\-     **ETA**: estimated time of arrival

\-    **ETD**: estimated time of departure

\-     **Amount**: auto calculated

\-     **Tax**: auto calculated

\-    **Discount amount**: auto calculated

<figure><img src="../../../.gitbook/assets/image (32).png" alt=""><figcaption></figcaption></figure>

\-   **Total amount**: auto calculated

If you’ve already have an excel file of item list, then

\-     Select **Template** button: to get the correct format. Add data by columns as the instructions.

\-     Select **Impor**t button: to upload the file. Go to the file location -> choose the file -> Open

\-     Select file name to import

#### 4.3.3                 Import the Purchase Order

To import the list of Purchase Orders:

\-     Select **Template**: to get the correct format file. Add data into Master and Detail sheet as the instructions.

\-     Select **Import**: to open Import Data window

\-     Select **Choose**: to upload the file, go to the file location -> choose the file -> Open

\-     Select **Upload**

\-     Check the error messages (if any) and correct them.

\-     **Upload Data**

#### 4.3.4                 Edit the Purchase Order.

You can edit/delete the Purchase Order detail when its status is **Open** by using the related icons (Shortcut and icons section). With Processing or Completed status, the data is locked. The Cancel one means the data is removed.

\-     To edit the Purchase Order, click the edit icon or PO number to open the its detail.

Adjust the General info directly and/or Detail info by selecting the edit icon in each item row.

Save.

\-     To delete a purchase order or an item, select the delete icon and follow the instruction.

In case you need to remove many items or many purchase orders, then check the boxes in front of each row -> Delete -> Save
