# 4.6 Return to Vendor

When the goods do not meet the standard quality, you ask the vendor to send it back to them. The Return to Vendor is created to record these incidents.

<figure><img src="../../../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure>

#### 4.6.1                   Search for a Return to Vendor

To open a return list, go to **Menu -> Purchase Order -> Return to Vendor.**

The window is opened as a grid table. You can select the filter button or give the condition to the

filter box of each field

<figure><img src="../../../.gitbook/assets/image (38).png" alt=""><figcaption></figcaption></figure>

#### 4.6.2                   Create a Return to Vendor

To create a return, select New to open the input form

<figure><img src="../../../.gitbook/assets/image (39).png" alt=""><figcaption></figcaption></figure>

_**General info**_

\-   **Reference No.**: auto created, unchangeable.

\-   **Receipt date**: created date

\-     **Vendor ID**: press F3 to get the dropdown list or type the code. Must fill to load the related receipt notes on the system.

\-    **Warehouse**:

\-    **Currency ID**:

\-     **% Discount**:

\-     **Note**

\-     **Description VN/EN/KR**:

\-     **Receipt Note**: press F3 to get the dropdown list.

\-    **Invoice Date (if any)**

\-   **Invoice No (if any)**

_**Detail info**_

When the **Receipt Note** is selected, the list of items will be \`loaded automatically. Select the item

codes that need to be returned on the receipt and adjust the quantity on the interface.

<figure><img src="../../../.gitbook/assets/image (40).png" alt=""><figcaption></figcaption></figure>

The user can also create item codes that are not associated with a **Receipt Note** by pressing the **New**

button to open the detail entry window

<figure><img src="../../../.gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure>

#### 4.6.3                   Edit Return to Vendor

Return to Vendor detail can be edited until it is on **Hold** status. When it is **Released,** its data will be linked to other modules and locked.

\-     To change the return to vendor detail, choose the return number or edit icon to open its detail.

\-     Change the general info and/or detail info. **Save.**

<figure><img src="../../../.gitbook/assets/image (42).png" alt=""><figcaption></figcaption></figure>

#### 4.6.4                   Delete/ Print the Return to Vendor

To delete a Return to Vendor, select delete icon and follow the instruction.

To delete many returns as one time, select the returns by checking the box -> Delete.

To print the return, click print icon.

To export the excel file, select the note(s) by checking the box(s) -> Export.
