# 3.2 Sales Price List

This part is the price information for customers. It is depended on the clients/ item and promotion seasons (if any)

<figure><img src="../../.gitbook/assets/image (74).png" alt=""><figcaption></figcaption></figure>

#### 3.2.1                   Create a new sales price list

Go to **Menu -> Sales Order -> Sales Price List** to open the list

<figure><img src="../../.gitbook/assets/image (75).png" alt=""><figcaption></figcaption></figure>

On the Price List screen, you can change the Price List Date to view older price lists (current date shown up as default).

You can also give the conditions on the filter box on the top of each field to search for any price lists

To create a new price list, select **New** to open the input form.

<figure><img src="../../.gitbook/assets/image (76).png" alt=""><figcaption></figcaption></figure>

_**General Info:**_

·     **Reference Number**: automatically create, no changeable. If you want to review created Price List, press F3 and choose the one you need.

·    **Price List Date**: the created date

·   **Customer code**: Press F3 to get the dropdown list of customers

·     **Validated date**: the date that price list is applied

·     **Expired date**: the date that price list is stopped using.

·     **Currency**: currency using in this price list (VND as default).

·     **Description VN/ EN/KR**: notes in languages.

Note: (\*) fields are must.

_**Detail Info (Item Detail)**_

On the Detail tab, click **New** to add items. The small Price form window pops up and you can follow the instructions.

·    **Item code**: press F3 to get the dropdown list of items or type the item code.

·     **Unit**: Automatically displayed based on the item code or press F3 to select the unit again.

·   **Unit Price**: if you choose foreign currency (not VND), it is automatically calculated.

·     **Quantity from**: the minimum order quantity to apply that price range

·     **Quantity to**: Maximum order quantity to apply that price range

·     **Tax code**: automatically loaded from Item list. Press F3 to change to different UOM

·     **Note.**

<figure><img src="../../.gitbook/assets/image (77).png" alt=""><figcaption></figcaption></figure>

If you have a list of items:

\-     Press **Template**: to get the excel form, add the data as the instruction.

\-     Press Import to upload file. Go to the file location -> choose the file -> Open. If the file is not read automatically, click the file name.

\-     Check the error column (if any) to adjust wrong format or wrong information.

\-    **Import/ Upload Data.**

#### 3.2.2                   Import Sales price list

If you have a list of sales price, then you can import the excel file as a faster way.

\-     Press **Template**: to get the excel format and fill in the necessary information by columns

\-     Press **Import**: to upload the file. Go to the file location -> choose the file -> Open.

\-     The file is uploaded automatically, if not click the file name

<figure><img src="../../.gitbook/assets/image (78).png" alt=""><figcaption></figcaption></figure>

\-     Check the **Error info** (if any) and adjust to the correct format/ information.

\-    **Upload data.**

<figure><img src="../../.gitbook/assets/image (79).png" alt=""><figcaption></figcaption></figure>

#### 3.2.3                   Edit the price list detail

When the price list has not used to make any orders yet, it is available to edit/delete its detail by using the related icons (Shortcut and Icons section)

**Note**: if the price list has already used, it does not allow to change any information.

To edit the price list, select edit icon or price list number.

<figure><img src="../../.gitbook/assets/image (80).png" alt=""><figcaption></figcaption></figure>

You can change the general info or detail info, similar to create a new price list. **Save**

#### 3.2.4                   Filter the prices by items/ by customers

If you need to review the prices by items (for all customers), go to **Menu -> Sales Order -> Price list detail**

<figure><img src="../../.gitbook/assets/image (81).png" alt=""><figcaption></figcaption></figure>

At the Item code column, choose the item code that you want to review (for one or all customers)
