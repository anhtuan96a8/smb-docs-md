# 3.7 Sales Return

When the goods do not meet the customer's standards, they may return the items. Sales Return will record information related to the return process.

<figure><img src="../../.gitbook/assets/image (16).png" alt=""><figcaption></figcaption></figure>

#### 3.7.1                  Create Sales Return

Go to **Menu -> Sales Order -> Sales Return** to open the list

Similar to the previous sections, the list of **Sales Return** orders is displayed in a table format, allowing the user to filter conditions for each column.

<figure><img src="../../.gitbook/assets/image (17).png" alt=""><figcaption></figcaption></figure>

To create Sales Return, Click **New**

<figure><img src="../../.gitbook/assets/image (18).png" alt=""><figcaption></figcaption></figure>

_**General info**_

\-     **Reference No**.: auto created, unchangeable.

\-     **Date**: created date

\-     **Customer code**: press F3 to get the dropdown list of customers or type the code. Must fill to load the related delivery notes

\-     **Warehouse**: warehouse to receive the returns.

\-    **Delivery No**.: related delivery note (press F3 for detail).

\-     **Note**

\-     **Description VN/EN/KR**

_**Detail info**_

When delivery note is chosen, the item list is loaded automatically.

\-     **Edit the item**: select edit icon to open the item detail -> **Save.**

\-     **Delete**: select delete icon to remove item -> **Save**.

\-     **Delete many items**: check the boxes -> Delete -> **Save.**

\-   **Print**: to review the Sales Return print form.



#### 3.7.2                   Edit Sales Return

Sales Return detail can be edited until it is on **Hold** status. When it is **Released**, its data will be linked to other modules and locked.

\-     To change the sales return detail, choose the return number or edit icon to open its detail.

\-     Change the general info or item info. **Save**.

#### 3.7.3                   Delete/ Print the Sales Return

To delete a Sales Return, select delete icon and follow the instruction.

To delete many Sales Return as one time, select the returns by checking the box -> Delete.

To print the Sales Return, click print icon.

To export the excel file, select the note(s) by checking the box(s) -> Export.

#### 3.7.4                   Release Sales Return

Release is a status that data is linked to other departments (accounting). At this stage, there is no edit/delete allow to make.
