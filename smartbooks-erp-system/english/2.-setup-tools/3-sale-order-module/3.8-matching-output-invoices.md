# 3.8 Matching output Invoices

When the delivery note is made, the invoices are also created to complete the sales documents. The invoice is updated by accounting team.

<figure><img src="../../.gitbook/assets/image (92).png" alt=""><figcaption></figcaption></figure>

**Matching Output Invoices**

Select **Menu → Sales Order → Matching Output Invoice** to open the list of **Sales Order** that have not been invoiced. The list will appear in a table format so users can filter conditions by column.

To change the display time, adjust the **From Date/To Date → View.**

<figure><img src="../../.gitbook/assets/image (93).png" alt=""><figcaption></figcaption></figure>

**To update the sales invoice:**

·     Check the sales orders that have not been invoiced.

·     Select the orders by ticking the box on the left side of each row →**Matching Output Invoices.**

<figure><img src="../../.gitbook/assets/image (94).png" alt=""><figcaption></figcaption></figure>
