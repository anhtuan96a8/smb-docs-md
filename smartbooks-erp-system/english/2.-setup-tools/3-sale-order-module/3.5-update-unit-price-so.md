# 3.5 Update unit price SO

**Update Unit Price SO**: This is used for updating the sales price for the last month after other

components have been costed. Sales orders that are updated with the new price are those that have

not been invoiced.

·     **Filter items by month and customer** → **Search**.

·     Proceed to update the price for the relevant sales orders.
