# 2.1 Change password

### 2.1.  Change password

To change user’s password, go to **Menu -> Setup Tools -> Change password**

<figure><img src="../.gitbook/assets/image (66).png" alt=""><figcaption></figcaption></figure>

On the Change Password window, fill in the Old password, then insert the New password twice. **Save.**
