# Phím tắt được sử dụng trên phần mềm

Để tối ưu hóa tốc độ nhập liệu và xử lý chứng từ, người dùng có thể áp dụng hệ thống phím tắt sau:

| Phím tắt     | Chức năng                                  |
|--------------|--------------------------------------------|
| F3           | Tìm kiếm                                   |
| Ctrl+N       | Tạo mới                                    |
| Ctrl+S       | Lưu                                        |
| Ctrl+D       | Xóa                                        |
| Ctrl+Q       | Đóng                                       |
| Ctrl+Shift+C | Sao chép chứng từ                          |
| Alt+D        | Sao chép dòng                              |
| F5           | Làm mới                                    |
| Alt+N        | Chứng từ tiếp theo                         |
| Alt+P        | Chứng từ trước                             |
| Alt+U        | Bỏ ghi sổ                                  |
| Ctrl+Enter   | Mở tra cứu chứng từ                        |
