# 14.2 Đăng nhập SmartBooks Website

Khi cần đăng nhập vào Cổng thông tin khách hàng trên nền web để tra cứu thông tin, báo cáo hoặc quản lý dữ liệu từ xa.

Để đăng nhập, người dùng thực hiện:

1. Truy cập website SmartBooks.
2. Nhập **Tên đăng nhập** hoặc **Địa chỉ email** đã đăng ký.
3. Nhập **Mật khẩu**.
4. Tích chọn **Ghi nhớ** nếu muốn đăng nhập tự động lần sau.
5. Nhấn **Đăng nhập**.

> **Lưu ý:** Nếu chưa có tài khoản, vui lòng liên hệ: [https://smartbooks.ssaudit.com/contact/](https://smartbooks.ssaudit.com/contact/)
