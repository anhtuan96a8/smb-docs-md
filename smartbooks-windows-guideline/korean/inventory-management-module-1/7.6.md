# 7.6	원가계산

**a) 재공품 마감잔액 및 환산량 (있는 경우) 수량을 입력합니다.**

재고 구성 메뉴에서 각 제품에 대해 계산기 수량 154를 선택합니다.

\-         **From date – To date:** 적용 기간

\-         **ProductID:** 기말 재공품 제품 코드

\-         **154Qty:** 기말 재공품 수량

\-         **621Rate, 622Rate, 627Rate:** 각 비용 그룹(621, 622, 627)에 대한 환산 비율

\-         **154EndAmt:** 원가 계산 완료 후 자동으로 업데이트되는 재공품 금액

\-         **621EndAmt, 622EndAmt, 627EndAmt:** 각 비용 그룹(621, 622, 627)의 금액도 원가 계산 완료 후 자동 업데이트 됩니다.

b) 원가 계산 (Costing)\
**1단계: 완제품 입고 수량 및 재공품 수량 계산**

* **ProductID:** 완제품 코드
* **FGQty:** 완제품 수량
* **154Qty:** 기말 재공품 수량 (원가 계산 전에 입력된 수량)
* **621Qty, 622Qty, 627Qty (재공 중 수량):** 154Qty × 621Rate, 622Rate, 627Rate
* **From Date – To Date**: 적용 기간
* **GroupID:** 제품 그룹 ID
* **PhaseID:** 공정 단계 ID

<figure><img src="../.gitbook/assets/image (73).png" alt=""><figcaption></figcaption></figure>

**2단계: 그룹별 완제품 총수량 계산**

1단계에서 산출한 완제품 수량을 바탕으로, 기간 내 각 그룹별 완제품 총수량을 계산합니다.

* **GroupID**: 완제품 그룹
* **621TotGoodQty, 622TotGoodQty, 627TotGoodQty** = FGQty(완제품 입고 수량) + 621Qty, 622Qty, 627Qty(재공 수량)
* **From Date – To Date**: 적용 기간
* **PhaseID:** 공정 단계 ID

<figure><img src="../.gitbook/assets/image (74).png" alt=""><figcaption></figcaption></figure>

기간별 Cost Center(원가 센터) 기준 비용 집계 요약

사용자가 621, 622, 627 계정에 비용을 입력할 때는 반드시 **Cost Center**를 선택해야 합니다.\
비용은 재고(Inventory) 및 \*\*일반전표(General Ledger)\*\*에서 입력됩니다.

* **CostCenterID:** 원가 센터 ID
* **CostMethodID, PhaseID, AllocationMethodID:** 사용자가 Cost Center 설정 시 구성한 비용 배분 방식에서 가져옵니다.
* 참고: **AllocationMethodID**는 CostMethodID가 2인 경우에만 사용됩니다.
* **Amt:** 해당 기간의 총비용
* **ProductID:** CostMethodID가 1인 경우, 제품별 직접비용을 입력할 때 사용됩니다.

<figure><img src="../.gitbook/assets/image (75).png" alt=""><figcaption></figcaption></figure>

PhaseID: 다음 기간에 사용할 공정 번호(Phase 번호)를 선택

Step 3, Step 4는 **비용 배분 기준이 수량 및 계수(coefficient)에 따른 방식**일 경우에 해당하며,\
즉, CostMethodID = 2, AllocationMethodID = 1로 설정된 Cost Center에 대한 배분에 사용됩니다.

한편, \*\*개별 식별 방식(Specific Identification)\*\*으로 설정된 Cost Center (CostMethodID = 1)이거나,\
\*\*원재료 기준 배분 방식(AllocationMethodID = 1)\*\*일 경우에는 Step 5에서 계산이 진행됩니다.

**3단계: 단계별로 제품 그룹에 대한 비용 배분 계산**

·        비용은 **Step 2**에서 설정된 `CostMethodID = 2` 및 `AllocationMethodID = 1`인 항목을 기준으로 집계됩니다.

·        `PhasedID`가 숫자인 경우, 전체 비용은 해당 Phase에 전부 배분됩니다.

·        `PhasedID`가 "all"인 경우, 전체 비용은 Phase별 배분 비율(Costing Rate Allocate Group)에 따라 배분됩니다.

비용 센터(CostCenter)별로 요약된 비용은 각 Phase 내에서 수량과 비율을 기준으로 제품 그룹에 배분되며, 결과는 `GroupAmt` 열에 표시됩니다.

·        **CostGroupID**: 비용 그룹 (621, 622, 627)

·        **CostCenterID**: 비용 센터 ID

·        **GroupID**: 제품 그룹 ID

·        **IsCosting**: 해당 제품 그룹에 비용 배분 여부 (1은 예, 0은 아니오)

·        **TotGoodQty**: Step 1.2에서 구한 그룹별 해당 기간 동안 완성된 제품 수량

o   `CostGroupID = 621`일 경우: `621TotGoodQty`에서 수량 참조

o   `CostGroupID = 622`일 경우: `622TotGoodQty`에서 수량 참조

o   `CostGroupID = 627`일 경우: `627TotGoodQty`에서 수량 참조

·        **GroupRate**: 제품 그룹별 배분 비율

·        **GroupQty**: 비용 센터별로 계산된 `TotGoodQty * GroupRate`

·        **CostCenterAmt**: 해당 Phase의 비용 센터별 총 비용

·        **CostCenterRate**: 제품 그룹별 비용 비율

o   계산식: `CostCenterRate = (TotGoodQty * GroupRate) / GroupQty`

·        **GroupAmt**: 각 비용 센터 및 제품 그룹에 배분된 총 비용

o   계산식: `GroupAmt = CostCenterAmt * CostCenterRate`

o   Step 4에서 사용 가능

·        **PhaseID**: 비용 배분 대상인 Phase

&#x20;         **AllocationMethodID**: 설정된 배분 방식 (항상 1로 설정됨)

<figure><img src="../.gitbook/assets/image (76).png" alt=""><figcaption></figcaption></figure>

**4단계: 제품별 비용 배분**

Step 3에서 산출된 제품 그룹별 총 비용(GroupAmt)을 기준으로, 개별 제품별로 비용을 배분합니다.\
결과는 `ProductAmt` 열에 표시됩니다.

·        **CostCenterID**: 비용 센터 ID

·        **GroupID**: 제품 그룹 ID

·        **GroupAmt**: 제품 그룹별 총 비용

·        **ProductID**: 해당 기간의 제품

·        **TotGoodQty**: 해당 기간의 제품 총 수량

o   계산식: **투입 수량(FGQty) + 재공품 수량(621Qty, 622Qty, 627Qty)**

·        **ProductRate**: 각 제품에 대한 배분 비율

·        **IsCosting**: 해당 제품에 대해 비용 배분 여부 (기본값은 1)

·        **TotProductAmt**: 제품 그룹별로 계산된 `TotGoodQty * ProductRate`

·        **Rate**: 각 제품별 비용 배분 비율

o   계산식: `Rate = (TotGoodQty * ProductRate) / TotProductAmt`

·        **ProductAmt**: 각 제품에 배분된 비용

계산식: `ProductAmt = GroupAmt * Rate`

<figure><img src="../.gitbook/assets/image (77).png" alt=""><figcaption></figcaption></figure>

**5단계: 완제품 원가 계산**

각 제품에 대한 총 비용은 기초 재공품 잔액과 당기 발생 비용을 포함합니다.\
당기 비용에는 다음이 포함됩니다:

·        개별 식별된 비용(Expense identification)

·        원재료 기준 배분된 비용(Expense allocate by raw material)

·        Step 4에서 산출된 제품별 배분 금액(Product Amount)

\*\*제품 원가(Costing Product)\*\*는 다음과 같이 계산됩니다:

총비용 ÷ 당기 총수량

Product ID: 제품 ID

Qty: step 1에서 계산된 완제품 수량

**BeginAmt:** 전기에서 이월된 재공품 기초 잔액

**Amt621\_ide, Amt622\_ide, Amt627\_ide:** 그룹 621,622,627별 기간별 비용 식별입니다. Costcenter의 2단계에서 CostMethodID =1을 가져옵니다.

**Amt621, Amt622, Amt627:** 기간별 비용은 각 그룹 621,622,627에 대해 4단계에서 가져오고 원재료별로 비용을 할당합니다.

TotAmt: = 기초 금액 (재공품 기초잔액 요약비용) + Amt621\_ide + Amt622\_ide + Amt627\_ide + Amt621 + Amt622 + Amt627

**단가**: 제품 원가 계산 = 총 금액 / 수량

**기말 금액** : 기말 재공품 = 단가 \* 재공품 수량

완제품 입고 전표 (finish goods receipt entry)에 반영됩니다.
