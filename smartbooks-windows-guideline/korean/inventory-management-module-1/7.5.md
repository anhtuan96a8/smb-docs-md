# 7.5	단계별 원가계산

#### a)             Cost Center (원가센터), 단계별, 제품군별 원가 계산 설정

**1단계: 비용 코드 (cost center) 생성 및 해당 비용에 대한 배분 방식 설정**

메인 메뉴 => 기본 정보 => 비용 코드

<figure><img src="../.gitbook/assets/image (67).png" alt=""><figcaption></figcaption></figure>

비용 센터 ID

비용 센터 이름

비용 그룹: 621: 원재료, 622: 인건비, 627: 간접비.

비용 배분 설정:

* **PhaseID:** 각 공정 단계(Phase)에 대한 비용을 설정합니다. 단계는 1, 2, 3 등으로 설정하거나 전체 단계를 의미하는 "All"로 설정할 수 있습니다.
* **CostMethodID:** 제품 그룹 또는 제품별로 비용을 배분하는 방식입니다.\
  • 옵션 1: 직접 재료비 기준 배분\
  • 옵션 2: 비율(rate) 기준 배분\
  • 옵션 3: 사용자 정의 방식 배분
* **AllocationMethodID:** (CostMethodID가 옵션 2일 때만 사용)\
  • 옵션 1: 제품 수량 \* 비율(예: 판매가, 계수 방식, 단순 방식 등)에 따라 비용 배분\
  • 옵션 2: 원재료 기준으로 인건비 및 제조간접비 배분
* **AllocationPhaseID:** 단계별 비용 배분 방식 (PhaseID가 All인 경우에만 사용)\
  • 옵션 1: 각 공정 단계에 동일한 비율로 배분\
  • 옵션 2: 각 공정 단계 및 각 Cost Center별로 다른 비율로 배분

**2단계: 각 공정 단계별 비용 배분 비율 설정**\
먼저, 해당 기간 동안 집계된 비용을 각 공정 단계별로 배분합니다.

* **AllocationPhaseID**: 기본값은 옵션 1 (모든 CostCenterID에 사용)
* **PhaseID**: 공정 단계 번호
* **Rate**: 각 단계(PhaseID)별 배분 비율
* **FromDate - ToDate**: 적용 기간



<figure><img src="../.gitbook/assets/image (68).png" alt=""><figcaption></figcaption></figure>

**3단계: 제품 그룹별 배분 비율 설정**\
이 단계는 비용을 수량 × 비율 방식으로 배분할 때 적용됩니다\
(CostMethodID = 2, AllocationMethodID = 1인 경우).

먼저, 해당 기간의 총비용을 각 공정 단계(Phase)별로 배분한 후,\
각 공정 단계별로 다시 제품 그룹(Group)별로 배분합니다.

설정 항목:

* GroupID: 제품 그룹 코드
* Rate: 수량 기준으로 균등하게 배분할 경우, 모든 GroupID에 대해 1로 설정
* PhaseID: 공정 단계 번호
* FromDate - ToDate: 적용 기간

<figure><img src="../.gitbook/assets/image (69).png" alt=""><figcaption></figcaption></figure>

**4단계: 개별 제품별 배분 비율 설정**\
이 단계는 비용을 수량 × 비율 방식으로 배분할 때 적용됩니다\
(CostMethodID = 2, AllocationMethodID = 1인 경우).

먼저, 해당 기간의 총비용을 각 공정 단계(Phase)별로 배분한 후,\
각 공정 단계별로 그룹(Group)별로, 마지막으로 개별 제품(Product)별로 배분합니다.

**설정 항목:**

* ProductID: 제품 코드
* Rate: 계수 방식, 판매가 방식 또는 단순 방식에 따른 배분 비율
* PhaseID: 공정 단계 번호
* FromDate - ToDate: 적용 기간
