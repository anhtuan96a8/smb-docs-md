# 스마트북 시스템의 단축키 및 아이콘 설명

**단축키 설명**

**F3:** 찾기

**F5:** 새로고침

**CTRL + N:** 전표 신규작성

**CTRL + S:** 전표 저장

**CTRL + D:** 전표 삭제

**CTRL + Q:** 창 닫기

**CTRL + SHIFT + C:** 전표 복사

**ALT + D:** 전표(열) 복사

**ALT + N:** 다음 전표

**ALT + P:** 이전 전표

**ALT + U:** 되돌리기

**CTRL + ->:** 옆으로 이동(tab)

**CTRL + <-:** 앞으로 이동 (tab)

**CTRL + H:** 메인 메뉴로 이동
