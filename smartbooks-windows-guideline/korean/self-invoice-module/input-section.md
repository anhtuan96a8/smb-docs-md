# 8.2 입력

#### a) 출고서, 내부이전



![](<../.gitbook/assets/0 (81).png>)

내부 재고 이동의 경우에도 세금계산서를 발행해야 하며, 재고 모듈을 사용하지 않습니다. 사용자는 이 기능을 통해 세금계산서를 생성할 수 있습니다.

#### b) 세금계산서 내역

![](<../.gitbook/assets/1 (46).png>)



사용자는 외상매출금 또는 재고관리 모듈의 판매 섹션에서 세금계산서 정보를 입력합니다. 해당 모듈에서 출고 배치를 선택하면, 인쇄할 세금계산서 목록이 표시됩니다. 인쇄할 세금계산서를 선택하려면 PRINT 열에 체크한 후, DETAIL 버튼을 클릭하여 세금계산서를 인쇄합니다. 이미 인쇄된 세금계산서의 경우, 세금계산서 인쇄 버튼은 비활성화됩니다.

#### c) 세금계산서 확인

![](<../.gitbook/assets/2 (42).png>)

확인할 세금계산서 번호를 입력하여, 인쇄상태 및 정보를 확인합니다.
