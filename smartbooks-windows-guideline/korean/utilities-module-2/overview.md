# 14.1 소개

Customer Portal은 회사가 Smartbooks Accounting S/W를 통해 기록한 회계정보 데이터를 기반으로, 주요 재무정보 (매출현황 / 매입현황 / 거래처별 채권잔액 / 매입처별 채무잔액 / 현금 보유 현황 / 주요 재고 및 원가현황) 를 도표 및 요약자료로 가공, 정리하여 회사의 전략적 의사결정과 주요 관리포인트 설정에 도움이 되고자 작성한 웹기반 Portal입니다.

이를 통하여 각 법인의 관리자와 본사 입장에서 법인의 현 재무상태를 일목요연하게 실시간 체크할 수 있으며 주요 거래처별 채권 / 매입처별 채무의 관리를 효율적으로 진행할 수 있습니다.
