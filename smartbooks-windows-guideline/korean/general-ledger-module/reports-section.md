# 1.5 보고서

입력된 거래에 대해 아래와 같은 보고서 출력이 가능합니다.

#### a) 시산표:

* 조회할 회계기간을 입력 (기산일/마감일)
* 통화유형 : 조회할 통화를 선택 (Currency ID)
* Excel file이 필요할 경우, 내보내기(Export Excel) 선택
* 인쇄 전 미리보기를 원할 시 “인쇄” 버튼 선택
* “인쇄” 버튼을 선택하여 출력
* “닫기” 버튼을 선택하여 창에서 나가기

\+ 인쇄 미리보기 상세 내용

&#x20;데이터 시트에서 인쇄할 시산표 내용확인이 가능합니다.

![](<../.gitbook/assets/0 (121).png>)

* 데이터 시트 상세 내용

\+ 외상매출금(131계정) 또는 외상매입금(331계정)이 아닌 계정을 더블 클릭시, 해당 계정의 원장이 아래와 같이 미리보기로 나타납니다.

![](<../.gitbook/assets/1 (90).png>)

\+ 외상매출금(131계정) 또는 외상매입금(331계정)을 더블 클릭시, 아래의 항목과 같이 해당 항목 아래에 거래처별 내역이 붉은 글씨로 보여집니다.

![](<../.gitbook/assets/2 (72).png>)

\+ 버튼

&#x20;인쇄 : 시산표 출력

&#x20;Excel로 내보내기 : 화면의 데이터 형식을 Excel로 변환

\+ 예시 : 외상매입금(331계정)의 거래처별 내역과 함께 Excel로 변환할 때

&#x20;1단계 : 331계정을 더블클릭 -> 화면에 상세 내역이 나타납니다.

&#x20;2단계 : 그 상태에서 Excel로 내보내기를 클릭합니다.

![](<../.gitbook/assets/3 (38).png>)

#### b) 요약 총계정원장

* 조회할 회계기간을 선택 (기산일/마감일)
* 조회할 통화유형 선택
* 조회할 보고서의 계정유형을 선택

\+ 전체 (ALL)

\+ 계정과목 (F3에서 선택)

* “인쇄 미리보기”를 선택하여 확인
* “닫기” 버튼을 선택하여 나가기

![](<../.gitbook/assets/4 (27).png>)

* Excel로 내보낸 파일에서 아래의 내용을 확인하실 수 있습니다.

\+ DOCSMAP 시트: 계정과목 목록

\+ DOCSMAP 시트에서 계정과목을 클릭시 해당 계정원장으로 이동합니다.

\+ 해당 계정원장에서 B10셀의 “DOCSMAP”을 클릭하면 DOCSMAP 시트로 이동합니다.

#### c) 상세 총계정원장

<figure><img src="../.gitbook/assets/image (80).png" alt=""><figcaption></figcaption></figure>

* 조회할 회계기간을 선택 (기산일/마감일)
* 조회할 통화유형 선택
* 조회할 보고서의 계정유형을 선택

\+ 전체 (ALL)

\+ 계정과목 (F3에서 선택)

* “인쇄 미리보기”를 선택하여 확인
* “닫기” 버튼을 선택하여 나가기
* “기산일” 옆 “VND-USD” 에 체크박스를 선택할 시, 인쇄와 Excel 내보내기에서 VND와 USD가 함께 나타납니다
* Excel로 내보내기 옵션

\+ 전체 보고서 : 계정별 원장, 시산표, 손익계산서, 대차대조표가 포함됩니다.

Index 시트에서 계정과목을 클릭시 해당 계정원장으로 이동합니다. ![](<../.gitbook/assets/5 (32).png>)

\_ 시산표 (Trial Balance)\
\_ 재무상태표 (Balance Sheet)\
\_ 손익계산서 (Income Statement)\
\_ 총계정 상세내역 (General Account Detail)\
\_ 색인 시트 (Index DOCSMAP sheet): 총계정 상세 시트 이름과 동일한 계정 목록

* VND-USD: 보고서를 VND 및 USD 통화로 함께 엑셀로 내보냅니다.

\+ 메인계정 보고서 : 계정 그룹별(레벨 1 : 111,112…) 원장이 Excel로 변환됩니다.

#### d) 대차대조표

* 조회할 회계기간을 선택 (기산일/마감일)
* 조회할 보고서 “Excel로 내보내기” 선택시 Excel로 저장
* “인쇄 미리보기”를 선택하여 확인
* “인쇄” 버튼을 선택하여 출력
* 미리보기 :

![](<../.gitbook/assets/6 (32).png>)

\+ 옵션(Option) :

세부내역 : “세부내역” 버튼 클릭시 데이터 화면에 해당 잔액 유형(차변,대변)과 계정과목 코드가 나타납니다

<figure><img src="../.gitbook/assets/image (91).png" alt=""><figcaption></figcaption></figure>

\+ Excel로 내보내기 (Export Excel)

Detail 체크박스 선택시 -> 변환한 Excel에 상세내역이 나타납니다.

Detail 체크박스 미선택시 -> 변환한 Excel에 요약본이 나타납니다.

\+ 인쇄

상세내역 : “세부내역”버튼 클릭 후 “인쇄” 버튼 선택

요약 : “인쇄”버튼 선택

#### e) 손익계산서:

![](<../.gitbook/assets/9 (17).png>)

\+ Excel로 내보내기 (Export Excel)

Detail 체크박스 미선택시 -> 변환한 Excel에 요약본이 나타납니다.
