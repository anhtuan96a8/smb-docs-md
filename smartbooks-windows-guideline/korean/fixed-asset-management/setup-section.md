# 5.1 설정

고정자산 관리

고정자산 관리 모듈은 단기 및 장기 선급비용, 월별 감가상각 계상, 월별 선급비용 배부 등에 대한 업무를 지원합니다.

![](<../.gitbook/assets/0 (100).png>)

고정자산관리 모듈은 아래의 5가지 부문으로 구성됩니다. :

* **설정**: 고정자산 유형, 고정자산, 장기선급비용 등록
* **입력**: 고정자산 청산 및 재평가
* **전기**(Process) : 감가비용 및 당기 선급비용 계상
* **고정자산 조회** : 고정자산 내용 및 당기 비용 조회
* **보고서**

### 5.1         설정 <a href="#toc201565706" id="toc201565706"></a>

#### a) 고정자산 구분

: 고정자산 구분코드, 적요(VN, EN, KR), 및 계정과목 코드

![](<../.gitbook/assets/0 (84).png>)

#### b) 고정자산 및 감가상각 등록

: 구매한 고정자산을 등록하여 관리합니다.

![](<../.gitbook/assets/1 (56).png>)

* 고정자산 코드 입력
* 고정자산명 입력
* 매입가격 입력
* **기본정보** 탭: 고정자산 구분코드, 단위, 일련번호, 원산지, 구매일, 상표, 모델명 등
* **감가상각비** 탭: 취득원가, 감가상각누계액, 감가상각비, 감가상각비 배부율, 자산재평가액, 고정자산 처분이익, 고정자산 처분손실에 대한 설정
* **기타** 탭: 자산의 내용연수, 감가상각 시작일, 감가상각 종료일, 실제 감가상각 종료일, 실제 감가상각액, 실제 감가상각 기간(월)에 대한 설정
* **“Print”** 버튼 클릭시 아래와 같이 고정자산관리대장 출력이 가능합니다.
* 1단계 : 데이터 부분에서 출력할 고정자산 선택
* 2단계 : “Print”버튼으로 출력

<figure><img src="../.gitbook/assets/image (106).png" alt=""><figcaption></figcaption></figure>

기타 옵션:

* Excel 내보내기 (Export Excel): 고정자산 목록을 Excel 파일로 내보내기
* 가져오기 (Import): Excel 파일에서 Smartbooks로 고정자산 정보 불러오기

Template 버튼을 클릭하여 엑셀 템플릿 파일을 다운로드합니다.

1단계: Import 체크박스를 클릭합니다.\
2단계: 엑셀 파일 선택 버튼을 클릭하여 데이터를 입력한 엑셀 파일을 선택합니다.\
3단계: 가져오기 실행 버튼을 클릭하여 시스템으로 데이터를 불러옵니다.

주의: 엑셀 시트 이름은 변경하지 마십시오.

##

#### c) 선급비용 배분



회계기간 중 증가한 선급비용을 입력하려면 배분비용 양식을 클릭하십시오.

![](<../.gitbook/assets/3 (32).png>)

\-         일반 정보 탭: 비용 ID, 비용명, 수량, 취득원가, 배분 기간, 이미 배분된 기간, 배분된 금액, 배분 시작일, 배분 종료일, 마지막 배분일

\-         기본 정보 탭: 계정, 비용 계정 1, 비율 %, 비용 계정 2, 비율 %

\-         기타 버튼: 인쇄 – 배분 카드 (필요 시)

n  1단계: 그리드 데이터에서 고정자산 코드를 선택

n  2단계: 인쇄 버튼 클릭

\-         기타 옵션:\
Export Excel: 배분 리스트를 Excel 파일로 내보내기\
Import: Excel 파일에서 Smartbooks로 배분 정보 불러오기

\-         Template 버튼 클릭하여 Excel 템플릿 파일 다운로드

\-         1단계: Import 체크박스 선택

\-         2단계: 데이터를 기입한 Excel 파일 선택

\-         3단계: Import 버튼 클릭하여 시스템에 업로드

\-         주의: Excel 시트 이름은 변경하지 마세요.
