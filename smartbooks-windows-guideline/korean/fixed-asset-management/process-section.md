# 5.3	데이터처리

#### a) 감가상각비 계상

![](<../.gitbook/assets/0 (89).png>)

본 항목은 아래의 데이터 시트에 나타난 고정자산 목록에서 선택시 자동으로 감가상각비를 계산하여 대체합니다. 해당 월에 감가상각비 계상할 고정자산을 체크박스로 선택 후 “저장”버튼을 누릅니다.

#### b) 선급비용 계상

![](<../.gitbook/assets/1 (52).png>)

본 항목은 아래의 데이터 시트에 나타난 장기선급비용 목록에서 선택시 자동으로 당기 비용을 계산하여 대체합니다. 해당 월에 인식할 장기선급비용을 체크박스로 선택 후 “저장”버튼을 누릅니다.
