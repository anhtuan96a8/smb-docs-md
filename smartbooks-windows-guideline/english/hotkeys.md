# Hotkeys and icons are used on the Smartbooks System

**Hotkeys**

**F3:** Search

**F5:** Refresh

**CTRL + N:** Create new Voucher

**CTRL + S:** Save voucher

**CTRL + D:** Delete Voucher

**CTRL + Q:** Close form

**CTRL + SHIFT + C:** Copy voucher to new one

**ALT + D:** Copy row on data grid

**ALT + N:** Next Voucher

**ALT + P:** Previous Voucher

**ALT + U:** Unrelease

**CTRL + ->:** Next screen (tab)

**CTRL + <-:** Previous screen (tab)

**CTRL + H:** Back to Main Menu
