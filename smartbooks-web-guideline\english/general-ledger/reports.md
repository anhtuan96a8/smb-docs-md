# 1.5 Report

After the accounting data has been processed in the respective accounting subsystems, users can generate the following reports.

#### 1.5.1.    Trial Balance

·       Select the accounting period to be exported (From date… To date…)

·       Select the currency to be displayed (Currency ID)

·       Choose to view the report on screen, print it as a PDF, or export it to Excel

<figure><img src="../.gitbook/assets/image (65).png" alt=""><figcaption></figcaption></figure>

For accounts that are not receivable or payable accounts (131, 331, 138, 338), double-clicking on a summary data line will display the detailed ledger for that account

<figure><img src="../.gitbook/assets/image (66).png" alt=""><figcaption></figcaption></figure>

For receivable and payable accounts (131, 331, 138, 338), double-clicking will display the outstanding balance details by each related party.

<figure><img src="../.gitbook/assets/image (67).png" alt=""><figcaption></figcaption></figure>

In cases where you want to export the Trial Balance with detailed receivable and payable information by each related party, double-click the corresponding receivable or payable account (131, 331, 138, 338), then select **Export to Excel**

<figure><img src="../.gitbook/assets/image (68).png" alt=""><figcaption></figcaption></figure>

#### 1.5.2    Summary General Ledger

·       Select the accounting period to be exported (From date… To date…).

·       Select the currency to be displayed (Currency ID)

·       Select the type of account to be exported (Option)

&#x20;       o   All Account

&#x20;       o   By account (press F3 to select the account number)

Select **Print** to view it in PDF format or **Export to Excel**.

<figure><img src="../.gitbook/assets/image (69).png" alt=""><figcaption></figcaption></figure>

Export excel với lựa chọn tất cả các tài khoản (All):

·       Sheet DOCSMAP: Danh mục các tài khoản chi tiết có trên file excel báo cáo

·       Click vào tài khoản để kết nối đến sheet cần xem

·       Trên từng sheet excel chi tiết: click DOCSMAP (ô B10) để trở về sheet DOCSMAP.

#### 1.5.3    Ledger Account Listing Detail Report

·       Select the accounting period to be exported (From date… To date…)

·       Select the currency to be displayed (Currency ID)

·       Select the type of account to be exported (Option):

&#x20;       o   All Acount

&#x20;       o   By account (press F3 to select the account number)

·       Select the report type

Choose **Print** to view it in PDF format or **Export to Excel**

<figure><img src="../.gitbook/assets/image (70).png" alt=""><figcaption></figcaption></figure>

\-        When selecting **Export GLDetail Report**:

·       Sheet Index: Lists all detailed accounts included in the Excel report.

·       Click an account to navigate to the corresponding detail sheet

·       On each detailed Excel sheet: click **DOCSMAP** (cell B10) to return to the Sheet Index.

\-        If selecting **Export GLDetail Main Account**: displays the report by main account level.

\-        If selecting **Export Full Report**: similar to exporting GLDetail, but also includes the **Balance Sheet (BS)**, **Trial Balance (TB)**, and **Profit and Loss Statement (PL)**.

<figure><img src="../.gitbook/assets/image (71).png" alt=""><figcaption></figcaption></figure>

#### 1.5.4    Balance sheet

·       Select the accounting period to be exported (From date… To date…).

·       Select the currency to be displayed (Currency ID)

·       Choose to view the report on screen, print it as a PDF, or export it to Excel

<figure><img src="../.gitbook/assets/image (72).png" alt=""><figcaption></figcaption></figure>

**Preview button**: displays information showing which accounts the report indicators are sourced from.

<figure><img src="../.gitbook/assets/image (73).png" alt=""><figcaption></figcaption></figure>

Export to Excel button: exports the report to Excel with the following options:

·      **Detail**: exports an Excel file containing indicator information from detailed accounts.

·       **Summary**: exports an Excel file containing only summarized indicators.

**Print button**: prints the report.

·       If the summary data grid is displayed, the report will be printed with summarized figures.

·       If the detailed data grid is displayed, the report will be printed with detailed account information.

**Export to Excel button**: exports the report to Excel with the following options:

·       **Detail**: exports an Excel file containing indicator information from detailed accounts.

·       **Summary**: exports an Excel file containing only summarized indicators.

**Print button**: prints the report.

·       When the summary data grid is displayed, the report will be printed with summarized figures
