# 10.1 Input section

## BUDGET MODULE

From Main Menu, Budget module comprises two sections:

* **Input section:** Enter budget.
* **Report section:** Report on budget.

### Input Section

![](<../.gitbook/assets/0 (129).png>)

* The user creates a budget for the current year.
* Input description by VN, EN or KR
* Choose cost account code, Team/Project, Cost Center
* Input amount budget for 12 months.
* Click save
