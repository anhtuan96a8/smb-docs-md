# 8.1 Setup section

## SELF INVOICE MODULE

![](<../.gitbook/assets/0 (1).png>)

From Main Menu, Sales Invoice module comprises three sections:

* **Setup section:** Void invoice; missing, burned, teared invoice.
* **Input section:** List of the invoice, check invoice.
* **Report section:** Report on the invoice issued. Report on the void invoice, Report on missing, burned, teared invoice, Report on invoice usage status.

8.1 Setup section

#### a) Issue invoice

![](<../.gitbook/assets/0 (132).png>)

This function is used to register invoice information will using for issue

*
  * Invoice information:
  * Invoice Form
  * Invoice Serial
  * Quantity
  * From number
  * To Number
  * Print Company: company have a license for print invoice
  * Print Contract :
  * No: number of contracts
  * Date: contract date

#### b) Cancel invoice

![](<../.gitbook/assets/1 (69).png>)

This function is used to cancel issued invoices that are no longer in use.

User needs to enter information in full such as invoice cancellation date.

The company name, address, tax code.

Invoice cancellation method.

#### c) Lost Bad Invoices

![](<../.gitbook/assets/2 (74).png>)

This Report is used in case the invoice is missing, burned or torn.

Enter report number, report date.

The company name, address, tax code.

Explain the Reason for invoice missing, burned, torn.
